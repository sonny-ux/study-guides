Chapter 6

The Riemann-Darboux Integral

Rudin · PS8 Ex.1–8 · Spring 2026 · Due 04/22

01 The Big Picture

Main Idea
Chapter 5 gave you derivatives — the local, infinitesimal structure of a function. Chapter 6 asks the opposite question: given a function, can we reconstruct its "total accumulation" over an interval? Rudin builds the integral from scratch using the Darboux approach: approximate the area with rectangles from above and below, and squeeze until they meet.
The Core Logic
Upper sums decrease as partitions get finer. Lower sums increase. Integrability means these two monotone sequences converge to the same value. Once you see that, the whole chapter unfolds naturally.

Darboux vs. Riemann: Instead of picking arbitrary sample points (Riemann), Darboux uses the supremum and infimum of $f$ on each piece — the tallest and shortest possible rectangles. The two approaches are equivalent but Darboux is cleaner.

02 Partitions and Darboux Sums

Main Idea
A partition chops an interval into subintervals. On each piece, take the supremum (upper rectangle) and infimum (lower rectangle) of $f$. Sum them weighted by width. The gap between upper and lower sum measures how oscillatory $f$ is.
Definition — Partition and Darboux Sums
A partition of $[a,b]$ is a finite set $P = \{x_0, x_1, \ldots, x_n\}$ with $a = x_0 < x_1 < \cdots < x_n = b$.

On each subinterval $[x_{k-1}, x_k]$, define $M_k = \sup f$ and $m_k = \inf f$. Then: $$U(P, f) = \sum_{k=1}^n M_k(x_k - x_{k-1}), \qquad L(P, f) = \sum_{k=1}^n m_k(x_k - x_{k-1})$$ Always $L(P,f) \leq U(P,f)$ since $m_k \leq M_k$.
The Geometric Picture — What the Sums Actually Look Like
$U(P,f)$ is the area of a staircase of rectangles that lies above the graph of $f$ — on each piece $[x_{k-1},x_k]$, the rectangle has height $M_k = \sup f$, the tallest possible. $L(P,f)$ is a staircase lying below the graph — height $m_k = \inf f$, the shortest possible.

The true area under $f$ is trapped: $L(P,f) \leq \int_a^b f \leq U(P,f)$. If you want a better approximation, use a finer partition — more, thinner rectangles force $M_k$ and $m_k$ closer together on each subinterval. Integrability is the statement that you can make the gap $U-L$ as small as you like by refining $P$.

Why sup and inf instead of sample points? Darboux deliberately picks the worst-case heights (tallest above, shortest below), giving the tightest possible bounds without any arbitrary choice. Riemann's approach picks an arbitrary sample point in each subinterval — same limit, messier bookkeeping.
Worked Example — $f(x)=x$ on $[0,1]$
Uniform partition $P_n$: $x_k = k/n$. Since $f$ is increasing, $M_k = k/n$ and $m_k = (k-1)/n$. $$U(P_n) = \frac{1}{n^2}\sum_{k=1}^n k = \frac{n+1}{2n} \to \tfrac{1}{2}, \qquad L(P_n) = \frac{n-1}{2n} \to \tfrac{1}{2}$$ Gap $= 1/n \to 0$. So $\int_0^1 x\,dx = 1/2$.

03 Refinements and the Monotonicity Lemma

Main Idea
Adding points to a partition can only make upper sums smaller and lower sums larger — the approximations tighten. This monotonicity is the engine behind integrability.
Refinement Lemma (Rudin §6.4)
If $P^*$ is a refinement of $P$, then: $L(P, f) \leq L(P^*, f) \leq U(P^*, f) \leq U(P, f)$

Why: Adding one point replaces one term $M_k\Delta x_k$ with two terms whose $M$-values are each $\leq M_k$. The upper sum can only decrease. Symmetrically, the lower sum can only increase.
Corollary — Any lower sum ≤ any upper sum
For any two partitions $P_1$ and $P_2$: $L(P_1, f) \leq U(P_2, f)$.

Proof: Let $P^* = P_1 \cup P_2$. Then $L(P_1) \leq L(P^*) \leq U(P^*) \leq U(P_2)$. This is crucial: the supremum of all lower sums and the infimum of all upper sums are well-defined.

04 The Definition of Integrability

Definition — Darboux Integrability (Rudin §6.1)
The upper integral is $\overline{\int_a^b} f = \inf_P U(P,f)$ and the lower integral is $\underline{\int_a^b} f = \sup_P L(P,f)$. Always $\underline{\int} f \leq \overline{\int} f$.

$f$ is Darboux integrable ($f \in \mathscr{R}$) if $\overline{\int_a^b} f = \underline{\int_a^b} f$. The common value is $\int_a^b f(x)\,dx$.
The Riemann Criterion — the practical test (Rudin §6.6)
$f \in \mathscr{R}$ if and only if: for every $\varepsilon > 0$, there exists a partition $P$ such that $$U(P, f) - L(P, f) < \varepsilon$$ Every integrability proof works by constructing such a partition.

05 Which Functions Are Integrable?

Theorem — Continuous ⟹ Integrable (Rudin §6.8)
If $f: [a,b] \to \mathbb{R}$ is continuous, then $f \in \mathscr{R}$.

Proof sketch: By compactness, $f$ is uniformly continuous. Given $\varepsilon > 0$, choose $\delta$ so $|x-y|<\delta \Rightarrow |f(x)-f(y)| < \varepsilon/(b-a)$. Take $P$ with all $\Delta x_k < \delta$. Then $M_k - m_k \leq \varepsilon/(b-a)$, giving $U-L \leq \varepsilon$.
Theorem — Monotone ⟹ Integrable (Rudin §6.9)
If $f: [a,b] \to \mathbb{R}$ is monotone, then $f \in \mathscr{R}$.

Proof (increasing $f$): Uniform partition $P_n$: $U-L = \frac{b-a}{n}(f(b)-f(a)) \to 0$. Monotone functions can have jump discontinuities — they're still integrable.
Non-example — Dirichlet function is NOT integrable
Let $f(x) = 1$ if $x \in \mathbb{Q}$, $f(x) = 0$ otherwise. On any subinterval, $M_k = 1$ and $m_k = 0$ (both $\mathbb{Q}$ and irrationals are dense). So $U(P,f) = b-a$ and $L(P,f) = 0$ for every partition — the gap is always $b-a$. So $f \notin \mathscr{R}$.
Properties & FTC

Algebraic Properties & the Fundamental Theorem

Linearity, monotonicity, triangle inequality, FTC, integration by parts, substitution.

01 Algebraic Properties

Main Idea
The Riemann integral is a linear functional on the space of integrable functions: you can add under the integral and pull out constants. These require proof — they're not free.
Linearity (Rudin §6.12)
If $f, g \in \mathscr{R}[a,b]$ and $c \in \mathbb{R}$, then $(f+g), (cf) \in \mathscr{R}$ and: $$\int_a^b (f+g) = \int_a^b f + \int_a^b g, \qquad \int_a^b cf = c\int_a^b f$$ Proof idea: $\sup(f+g) \leq \sup f + \sup g$ gives $U(P,f+g) \leq U(P,f)+U(P,g)$. Similarly $L(P,f+g) \geq L(P,f)+L(P,g)$. These squeeze $\int(f+g)$ to equal $\int f + \int g$.
Monotonicity
If $f \leq g$ on $[a,b]$, then $\int_a^b f \leq \int_a^b g$.

Proof: $h = g-f \geq 0$ implies $m_k(h) \geq 0$, so $L(P,h) \geq 0$ for all $P$, giving $\int h \geq 0$.
Triangle Inequality for Integrals
If $f \in \mathscr{R}[a,b]$, then $|f| \in \mathscr{R}[a,b]$ and $\left|\int_a^b f\right| \leq \int_a^b |f|$.

Why $|f| \in \mathscr{R}$: $\big||f(x)|-|f(y)|\big| \leq |f(x)-f(y)|$, so oscillation of $|f|$ is no worse than $f$.
Why the bound: From $-|f| \leq f \leq |f|$, monotonicity gives $-\int|f| \leq \int f \leq \int|f|$.

02 Fundamental Theorem of Calculus

Main Idea
The FTC reveals that integration and differentiation are inverse operations. Part 1: integrating produces a differentiable function — starting from $f$, you build $F(x)=\int_a^x f$ and recover $f$ by differentiating $F$. Part 2: you can evaluate integrals via antiderivatives — if you already know a function $F$ with $F'=f$, then $\int_a^b f = F(b)-F(a)$.
Geometric Meaning — Why $F'(x_0) = f(x_0)$
$F(x)=\int_a^x f(t)\,dt$ is the signed area under the graph of $f$ from $a$ to $x$ — a running total. When $x$ increases by a small amount $h$, the new area added is $\int_{x_0}^{x_0+h}f(t)\,dt$, which is approximately a thin rectangle of width $h$ and height $f(x_0)$, giving area $\approx f(x_0)\cdot h$. So: $$\frac{F(x_0+h)-F(x_0)}{h}\approx\frac{f(x_0)\cdot h}{h}=f(x_0)$$ The approximation becomes exact as $h\to 0$, because continuity of $f$ at $x_0$ means the height of $f$ in $[x_0,x_0+h]$ is close to $f(x_0)$ for small $h$. This is why the continuity hypothesis on $f$ is necessary — without it, the "height of the thin strip" isn't well-defined, and $F$ may fail to be differentiable at that point.

Geometric summary of Part 2: If $F$ is any antiderivative of $f$, then $F(b)-F(a)$ is the net change in $F$ over $[a,b]$, which equals the total signed area $\int_a^b f$. This works because $F$ and the Part 1 integral function $\int_a^x f$ differ by a constant (both have the same derivative $f$).
FTC Part 1 — The integral function is differentiable (Rudin §6.20)
Let $f \in \mathscr{R}[a,b]$ and define $F(x) = \int_a^x f(t)\,dt$. Then:
· $F$ is continuous on $[a,b]$.
· If $f$ is continuous at $x_0$, then $F'(x_0) = f(x_0)$.

Proof sketch: $\frac{F(x_0+h)-F(x_0)}{h} - f(x_0) = \frac{1}{h}\int_{x_0}^{x_0+h}(f(t)-f(x_0))\,dt$. By continuity of $f$ at $x_0$, the integrand is $< \varepsilon$ for $|h| < \delta$, so the whole expression is $< \varepsilon$.
FTC Part 2 — Evaluation using antiderivatives (Rudin §6.21)
If $f \in \mathscr{R}[a,b]$ and $F: [a,b] \to \mathbb{R}$ is continuous with $F' = f$ on $(a,b)$, then: $$\int_a^b f(x)\,dx = F(b) - F(a)$$ Reading this: FTC Part 2 applies to ANY antiderivative $F$, not just the one defined by Part 1. This is what makes it useful for computation.
Common error on FTC Part 1
FTC Part 1 does NOT say $F'(x_0) = f(x_0)$ for all $x_0$ — only at points where $f$ is continuous. If $f$ has a jump at $x_0$, $F$ is still continuous there but may fail to be differentiable. $f \in \mathscr{R}$ is not enough to guarantee differentiability of $F$.

03 Integration by Parts & Substitution

Integration by Parts (Rudin §6.22)
Let $u, v: [a,b] \to \mathbb{R}$ be continuously differentiable. Then: $$\int_a^b u'(x)v(x)\,dx = u(b)v(b) - u(a)v(a) - \int_a^b u(x)v'(x)\,dx$$ Proof: Apply FTC to the product rule $(uv)' = u'v + uv'$, then rearrange.
Integration by Substitution (Rudin §6.19)
Let $\varphi: [a,b] \to \mathbb{R}$ be $C^1$ and $f$ continuous on the range of $\varphi$. Then: $$\int_a^b f(\varphi(x))\,\varphi'(x)\,dx = \int_{\varphi(a)}^{\varphi(b)} f(t)\,dt$$ Proof: Let $F$ be an antiderivative of $f$. Chain rule gives $(F\circ\varphi)' = f(\varphi(x))\varphi'(x)$. Apply FTC Part 2 to both sides.
Exam trap — hypotheses matter
Substitution requires $f$ continuous on the range of $\varphi$ AND $\varphi$ continuously differentiable — not just differentiable. Always state these hypotheses explicitly when writing proofs.

Measure Zero, Hölder, Arc Length

01 Measure Zero and Lebesgue's Criterion

Main Idea
The exact boundary of Riemann integrability: a bounded function is integrable if and only if its set of discontinuities has measure zero. This unifies and explains all the integrability theorems.
Definition — Measure Zero (PS8 Ex.4)
$A \subseteq \mathbb{R}$ has measure zero if for every $\varepsilon > 0$, there exist open intervals $\{O_n\}$ with $A \subseteq \bigcup O_n$ and $\sum \ell(O_n) < \varepsilon$.

Finite sets are measure zero. Countable sets are measure zero. The Cantor set is measure zero despite being uncountable.
Lebesgue's Criterion — complete characterization (PS8 Ex.4)
A bounded $f: [a,b] \to \mathbb{R}$ is Darboux integrable $\iff$ the set of discontinuities of $f$ has measure zero.

Why this matters: Explains why continuous $f$ (zero discontinuities) and monotone $f$ (at most countably many discontinuities = measure zero) are integrable, while the Dirichlet function (discontinuous everywhere — discontinuity set $= [a,b]$, measure $b-a > 0$) is not.
Proof strategy for PS8 Ex.4
⇒
Define $D_\alpha = \{x : \omega(f,x) \geq \alpha\}$ where $\omega(f,x)$ is the oscillation at $x$. Show $D_\alpha$ is compact. By the Riemann criterion, choose $P$ with $U-L < \alpha\varepsilon$. Intervals meeting $D_\alpha$ have $M_k - m_k \geq \alpha$, so their total length $\leq \varepsilon$.
Then: $D = \bigcup D_{1/n}$ is a countable union of measure-zero sets, hence measure zero.
⇐
Cover discontinuity set $D$ by open intervals of total length $< \varepsilon/(4M)$. Let $K = [a,b] \setminus \text{cover}$. $K$ is compact and $f|_K$ is uniformly continuous. Handle $K$ by uniform continuity; handle the covering by bounding oscillation by $2M$. Combine to get $U-L < \varepsilon$.
Key: splitting into "good" (near $K$) and "bad" (near $D$) subintervals.

02 Hölder's Inequality

Hölder's Inequality (PS8 Ex.6)
Let $f, g \in \mathscr{R}[a,b]$, $p, q > 1$ with $\frac{1}{p}+\frac{1}{q}=1$. Then: $$\left|\int_a^b fg\,dx\right| \leq \left(\int_a^b |f|^p\right)^{1/p}\left(\int_a^b |g|^q\right)^{1/q}$$ The case $p = q = 2$ is the Cauchy-Schwarz inequality.
Proof — via Young's inequality
1
Young's inequality: $ab \leq a^p/p + b^q/q$ for $a,b \geq 0$.
Proof: fix $b$, minimize $g(a) = a^p/p + b^q/q - ab$ over $a \geq 0$. The minimum is $0$.
2
Let $A = \|f\|_p$ and $B = \|g\|_q$. If either is $0$, done. Otherwise normalize: $\tilde{f} = |f|/A$, $\tilde{g} = |g|/B$, so $\int\tilde{f}^p = \int\tilde{g}^q = 1$.
This reduces to the case where both $L^p$-norms equal $1$.
3
Apply Young's pointwise: $\tilde{f}(x)\tilde{g}(x) \leq \tilde{f}^p/p + \tilde{g}^q/q$. Integrate: $\int\tilde{f}\tilde{g} \leq 1/p + 1/q = 1$. Unscale: $\int|fg| \leq AB$. Then $|\int fg| \leq \int|fg| \leq AB$.

03 Arc Length

Definition — Arc Length (PS8 Ex.7)
$$\mathcal{L}(f) = \sup_{P} \sum_{k=1}^n \sqrt{(x_k - x_{k-1})^2 + (f(x_k) - f(x_{k-1}))^2}$$
Arc Length Formula for $C^1$ functions
If $f \in C^1[a,b]$, then $\mathcal{L}(f) = \int_a^b\sqrt{1+f'(x)^2}\,dx$.

Proof sketch: MVT gives $f(x_k)-f(x_{k-1}) = f'(c_k)\Delta x_k$. So each segment length $= \sqrt{1+f'(c_k)^2}\,\Delta x_k$ — a Riemann sum for the integral. Uniform continuity of $f'$ ensures convergence.
Application — Circumference $= 2\pi r$
Upper semicircle: $f(x) = \sqrt{r^2-x^2}$, $f'(x) = -x/\sqrt{r^2-x^2}$, $1+f'^2 = r^2/(r^2-x^2)$. $$\int_{-r}^r \frac{r}{\sqrt{r^2-x^2}}\,dx = r\left[\arcsin\frac{x}{r}\right]_{-r}^r = r\pi$$ Total $= 2\pi r$.

04 Fundamental Lemma of the Calculus of Variations

Fundamental Lemma (PS8 Ex.8)
Let $f: [a,b] \to \mathbb{R}$ be continuous. If $\int_a^b f(x)\phi(x)\,dx = 0$ for every continuous $\phi: [a,b] \to \mathbb{R}$, then $f \equiv 0$.
Proof by contradiction
1
Suppose $f(x_0) > 0$ for some $x_0$. By continuity, $f > 0$ on some interval $I = (x_0-\delta, x_0+\delta) \cap [a,b]$.
2
Choose a bump function: $\phi(x) = \max(0, \delta - |x-x_0|)$ (a tent function). This is continuous, zero outside $I$, positive inside $I$.
Test function technique — ubiquitous in PDEs and distribution theory.
3
Then $f(x)\phi(x) > 0$ on $I$ and $\geq 0$ elsewhere. So $\int_a^b f\phi > 0$ — contradiction. $\blacksquare$

PS8 Ex.1Let $c > 0$. Evaluate $\int_0^c x\,dx$ using Darboux sums, without using FTC.PS8
Approach
Take the uniform partition $P_n$. Since $f(x)=x$ is increasing, $M_k = kc/n$ and $m_k = (k-1)c/n$. Use $\sum_{k=1}^n k = n(n+1)/2$. Show both $U$ and $L$ converge to $c^2/2$.
Solution
$U(P_n) = \frac{c^2}{n^2}\sum_{k=1}^n k = \frac{c^2(n+1)}{2n} = \frac{c^2}{2}+\frac{c^2}{2n}$

$L(P_n) = \frac{c^2(n-1)}{2n} = \frac{c^2}{2}-\frac{c^2}{2n}$

Gap $= c^2/n \to 0$. Since $L(P_n) \leq \underline{\int} \leq \overline{\int} \leq U(P_n)$ and both bounds $\to c^2/2$: $\int_0^c x\,dx = c^2/2$.
PS8 Ex.2Let $f,g \in \mathscr{R}[a,b]$. Prove linearity, monotonicity, and the triangle inequality for the Riemann integral.PS8
Approach
Additivity: $\sup(f+g) \leq \sup f + \sup g$ gives $U(P,f+g) \leq U(P,f)+U(P,g)$. Squeeze.
Scalar: Handle $c \geq 0$ and $c < 0$ separately ($\sup(cf) = c\inf f$ when $c < 0$).
Monotonicity: $h = g-f \geq 0$ implies $L(P,h) \geq 0$ for all $P$.
Triangle: Show $|f| \in \mathscr{R}$ via reverse triangle inequality, then apply $-|f| \leq f \leq |f|$.
Solution sketch
Part 1 (Additivity): $U(P,f+g) \leq U(P,f)+U(P,g)$ and $L(P,f+g) \geq L(P,f)+L(P,g)$. Given $\varepsilon > 0$, choose $P$ so each of $U(P,f), U(P,g)$ is within $\varepsilon/2$ of its integral. Then $U(P,f+g) \leq \int f + \int g + \varepsilon$. Similarly from below. Squeeze gives $\int(f+g) = \int f + \int g$.

Part 2: For $c \geq 0$: $\sup(cf) = c\sup f$, so $U(P,cf) = cU(P,f)$. For $c < 0$: $\sup(cf) = c\inf f = |c|\cdot(-\inf f)$, giving $U(P,cf) = -|c|L(P,f)$. Infimize/supremize to get $c\int f$ in both cases.

Part 3: $h=g-f \geq 0$: on each subinterval $m_k(h) \geq 0$, so $\int h \geq 0$, i.e., $\int g \geq \int f$.

Part 4: $|f| \in \mathscr{R}$: $\sup|f|-\inf|f| \leq \sup f - \inf f$ by reverse triangle inequality, so $U(P,|f|)-L(P,|f|) \leq U(P,f)-L(P,f) \to 0$. Then $-|f| \leq f \leq |f|$ gives $-\int|f| \leq \int f \leq \int|f|$. $\square$
PS8 Ex.3Prove that integration by parts and integration by substitution hold.PS8
IBP
$(uv)' = u'v + uv'$. Integrate over $[a,b]$ via FTC: $u(b)v(b)-u(a)v(a) = \int u'v + \int uv'$. Rearrange. $\square$
Substitution
Let $F$ be an antiderivative of $f$ (exists by FTC Part 1, $f$ continuous). Chain rule: $(F\circ\varphi)' = f(\varphi(x))\varphi'(x)$. By FTC Part 2: $\int_a^b f(\varphi(x))\varphi'(x)\,dx = F(\varphi(b))-F(\varphi(a)) = \int_{\varphi(a)}^{\varphi(b)} f(t)\,dt$. $\square$
PS8 Ex.4Prove Lebesgue's criterion: $f \in \mathscr{R}[a,b]$ iff the discontinuity set of $f$ has measure zero.PS8
$(\Rightarrow)$ Integrability implies measure-zero discontinuities
Fix $\alpha > 0$. Let $D_\alpha = \{x : \omega(f,x) \geq \alpha\}$. Note $D_\alpha$ is closed (upper semi-continuity of oscillation), hence compact. Given $\varepsilon > 0$, the Riemann criterion gives $P$ with $U-L < \alpha\varepsilon$. Intervals of $P$ meeting $D_\alpha$ satisfy $M_k-m_k \geq \alpha$, so their total length $\leq \varepsilon$. Thus $D_\alpha$ has measure zero. $D = \bigcup_{n=1}^\infty D_{1/n}$ is a countable union, hence measure zero. $\square$
$(\Leftarrow)$ Measure-zero discontinuities implies integrability
Let $M = \sup|f|$. Given $\varepsilon > 0$, cover discontinuity set $D$ by open intervals $\{O_j\}$ with $\sum\ell(O_j) < \varepsilon/(4M+1)$. Let $K = [a,b] \setminus \bigcup O_j$. $K$ is compact, $f|_K$ is uniformly continuous: choose $\delta$ so $|f(x)-f(y)| < \varepsilon/(2(b-a))$ on $K$ for $|x-y| < \delta$. Take partition $P$ with $\|P\| < \delta$ and all $O_j$-endpoints included. "Good" intervals (in $K$): oscillation $< \varepsilon/(2(b-a))$, contributing $\leq \varepsilon/2$. "Bad" intervals (in some $O_j$): oscillation $\leq 2M$, total length $< \varepsilon/(4M)$, contributing $< \varepsilon/2$. Total: $U-L < \varepsilon$. $\square$

PS8 Ex.6Prove Hölder's inequality: $|\int fg| \leq \|f\|_p \|g\|_q$ for conjugate exponents $p, q > 1$.PS8
Solution
Young's: $ab \leq a^p/p + b^q/q$ for $a,b \geq 0$. (Minimize $a^p/p + b^q/q - ab$ over $a$; minimum is $0$.)

Normalize: Let $A = \|f\|_p$, $B = \|g\|_q$. If either is $0$: both sides are $0$. Otherwise set $\tilde{f} = |f|/A$, $\tilde{g} = |g|/B$, so $\int\tilde{f}^p = \int\tilde{g}^q = 1$.

PS8 Ex.7Prove the arc length formula $\mathcal{L}(f) = \int_a^b\sqrt{1+f'(x)^2}\,dx$ and use it to show circumference $= 2\pi r$.PS8
Arc length formula
For partition $P$: $S(P) = \sum\sqrt{\Delta x_k^2 + \Delta f_k^2}$. By MVT, $\Delta f_k = f'(c_k)\Delta x_k$, so $S(P) = \sum\sqrt{1+f'(c_k)^2}\,\Delta x_k$ — a Riemann sum for $\int_a^b\sqrt{1+f'^2}$. Since $f' \in C[a,b]$, $\sqrt{1+f'^2}$ is continuous, so Riemann sums converge to the integral. The sup of $S(P)$ equals this integral. $\square$

$f(x) = \sqrt{r^2-x^2}$: $1+f'^2 = r^2/(r^2-x^2)$. Arc length of upper semicircle: $$\int_{-r}^r \frac{r}{\sqrt{r^2-x^2}}\,dx = r[\arcsin(x/r)]_{-r}^r = r\pi$$ Double for full circle: $2\pi r$. $\square$
PS8 Ex.8Let $f$ be continuous on $[a,b]$ with $\int_a^b f\phi = 0$ for every continuous $\phi$. Show $f \equiv 0$.PS8
Solution
Suppose $f(x_0) > 0$. By continuity, $f > 0$ on some interval $I = (x_0-\delta, x_0+\delta) \cap [a,b]$. Define the tent function $\phi(x) = \max(0, \delta - |x-x_0|)$. This is continuous, $\phi > 0$ on $I$, and $\phi = 0$ outside $I$. Then $f\phi > 0$ on $I$ and $\geq 0$ elsewhere, so $\int_a^b f\phi > 0$ — contradiction. $\square$

Rudin 6.1Suppose $f \in \mathscr{R}$ on $[a,b]$, $\alpha \leq f \leq \beta$, $\phi$ is continuous on $[\alpha,\beta]$, and $h = \phi\circ f$. Prove $h \in \mathscr{R}$.RUDIN
Solution
$\phi$ is uniformly continuous on $[\alpha,\beta]$: for $\varepsilon > 0$, get $\eta > 0$ with $|s-t| < \eta \Rightarrow |\phi(s)-\phi(t)| < \varepsilon/(2(b-a))$. Let $M = \sup|\phi|$. Since $f \in \mathscr{R}$, choose $P$ with $U(P,f)-L(P,f) < \eta\varepsilon/(4M)$.

Split subintervals: good ($M_k(f)-m_k(f) < \eta$) contribute $\leq \varepsilon/2$ to $U(P,h)-L(P,h)$ by $|\phi$ oscillation $| \leq \varepsilon/(2(b-a))$. Bad intervals have total length $\leq U(P,f)-L(P,f))/\eta < \varepsilon/(4M)$, contributing $\leq 2M\cdot\varepsilon/(4M) = \varepsilon/2$. Total $< \varepsilon$. $\square$
Rudin 6.2Suppose $f, g \in \mathscr{R}$ on $[a,b]$. Prove that $fg \in \mathscr{R}$.RUDIN
Solution
Use the identity $fg = \frac{1}{4}[(f+g)^2-(f-g)^2]$. By linearity, $f+g$ and $f-g$ are integrable. Apply Rudin 6.1 with $\phi(t) = t^2$ (continuous on any bounded interval): $(f+g)^2$ and $(f-g)^2$ are integrable. By linearity, $fg \in \mathscr{R}$. $\square$
Rudin 6.10Let $p \geq 0$, $p \in \mathscr{R}$, and $f$ continuous on $[a,b]$. Show $\exists\,c \in [a,b]$ with $\int_a^b fp = f(c)\int_a^b p$.RUDIN
Solution
Let $m = \min f$, $M = \max f$ (attained since $f$ continuous on compact $[a,b]$). Since $p \geq 0$ and $m \leq f \leq M$: $m\int p \leq \int fp \leq M\int p$.

If $\int p = 0$: then $p \equiv 0$ (since $p \geq 0$), so $\int fp = 0 = f(c)\int p$ for any $c$. If $\int p > 0$: $m \leq \frac{\int fp}{\int p} \leq M$. By IVT applied to $f$, $\exists\,c$ with $f(c) = \frac{\int fp}{\int p}$. $\square$

Quick Drills

True/false, proof ordering, and multiple choice. Tap to answer — explanation reveals immediately.

01 True / False

T/FEvery continuous function on $[a,b]$ is Riemann integrable.
True — Rudin Thm 6.8
Continuity on compact $[a,b]$ gives uniform continuity (Thm 4.19). Uniform continuity makes the oscillation $M_k - m_k$ uniformly small on all subintervals — exactly what the Riemann criterion requires.
T/FIf $f \in \mathscr{R}[a,b]$, then $f$ is continuous on $[a,b]$.
False — converse of Thm 6.8 fails
Any monotone function is integrable but can have jump discontinuities. More generally, bounded functions with measure-zero discontinuity sets are integrable. The Dirichlet function $\mathbf{1}_\mathbb{Q}$ is bounded with dense discontinuities — NOT integrable ($U=1$, $L=0$).
T/FFTC Part 1: if $F(x) = \int_a^x f(t)\,dt$ and $f \in \mathscr{R}[a,b]$, then $F'(x) = f(x)$ for all $x \in [a,b]$.
False — only where $f$ is continuous (Rudin §6.20)
$F$ is always continuous (Lipschitz, since $f$ is bounded). But $F'(x_0) = f(x_0)$ holds only at points where $f$ is continuous. If $f$ has a jump at $x_0$, $F$ may fail to be differentiable there.
T/FA refinement of a partition $P$ increases $U(f,P)$ and decreases $L(f,P)$.
False — it's the opposite direction (Rudin Thm 6.4)
Refinement decreases $U$ and increases $L$. Finer subintervals give tighter sup/inf on each piece. Monotonicity lemma: $L(P) \leq L(P^*) \leq U(P^*) \leq U(P)$ for any refinement $P^*$.
T/FIf $f$ is monotone on $[a,b]$, then $f \in \mathscr{R}[a,b]$.
True — Rudin Thm 6.9
Uniform partition of $n$ pieces: $U-L = \frac{b-a}{n}(f(b)-f(a)) \to 0$. Regardless of how many jump discontinuities $f$ has, the total variation $f(b)-f(a)$ is bounded and can be spread over $n$ pieces.
T/FFor any bounded $f:[a,b]\to\mathbb{R}$ and any partition $P$: $L(f,P) \leq U(f,P)$.
True — always, no integrability required
On each subinterval: $m_k = \inf f \leq \sup f = M_k$, so $\sum m_k\Delta x_k \leq \sum M_k\Delta x_k$. This holds for any bounded $f$ and any partition. The stronger statement: $L(f,P_1) \leq U(f,P_2)$ for ANY two partitions (proved via common refinement).

⠿BNormalize: set $\tilde{f} = |f|/\|f\|_p$ and $\tilde{g} = |g|/\|g\|_q$, so $\int\tilde{f}^p = \int\tilde{g}^q = 1$.

A: Integrate and use $1/p+1/q=1$ — the normalization was exactly designed so both integrals are $1$.
C: Undo the normalization to get the general statement.
ORDERRudin 6.1Drag to arrange: prove that if $f \in \mathscr{R}[a,b]$ and $\phi$ is continuous on $[\alpha,\beta] \supseteq f([a,b])$, then $h = \phi \circ f \in \mathscr{R}[a,b]$.
⠿AOn good intervals ($M_k(f)-m_k(f) < \eta$): $M_k(h)-m_k(h) < \varepsilon/(2(b-a))$, so their total contribution to $U(P,h)-L(P,h)$ is $\leq \varepsilon/2$.
⠿BSince $f \in \mathscr{R}$, choose a partition $P$ with $U(P,f)-L(P,f) < \eta\varepsilon/(4M)$ where $M = \sup|\phi|$.
⠿CCombining: $U(P,h)-L(P,h) \leq \varepsilon/2 + \varepsilon/2 = \varepsilon$. So $h \in \mathscr{R}$ by the Riemann criterion. $\blacksquare$
⠿DSince $\phi$ is uniformly continuous on compact $[\alpha,\beta]$: get $\eta > 0$ with $|s-t| < \eta \Rightarrow |\phi(s)-\phi(t)| < \varepsilon/(2(b-a))$.
⠿EOn bad intervals ($M_k(f)-m_k(f) \geq \eta$): total length $< \varepsilon/(4M)$, oscillation of $h$ is $\leq 2M$, so their contribution is $< \varepsilon/2$.
⠿FCall subinterval $k$ good if $M_k(f)-m_k(f) < \eta$, and bad otherwise.
Correct order: D → B → F → A → E → C
D: Uniform continuity of $\phi$ gives the $\eta$ that controls oscillation of $h$ on good intervals.
B: Use $f \in \mathscr{R}$ to get a partition where total oscillation of $f$ is small — the threshold $\eta\varepsilon/(4M)$ is chosen to make bad intervals short.
F: Set up the good/bad dichotomy — the central structural idea of the proof.
A: Good intervals: $f$ barely oscillates $\Rightarrow$ $h$ barely oscillates (via $\eta$ from D).
E: Bad intervals: $f$ oscillates a lot, but compactness of $P$ forces their total length to be tiny.
C: Add the two halves and invoke the Riemann criterion.

MULTIThe Riemann criterion says $f \in \mathscr{R}[a,b]$ if and only if:
B — Rudin §6.6
