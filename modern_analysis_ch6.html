01 Partitions and Darboux Sums

Definition 6.1 — Partition
A partition $P$ of $[a,b]$ is a finite set $\{x_0, x_1,\ldots,x_n\}$ with $a = x_0 < x_1 < \cdots < x_n = b$. Write $\Delta x_k = x_k - x_{k-1}$.

On each $[x_{k-1},x_k]$ define $M_k = \sup f$ and $m_k = \inf f$. The upper and lower Darboux sums are: $$U(P,f) = \sum_{k=1}^n M_k\,\Delta x_k \qquad L(P,f) = \sum_{k=1}^n m_k\,\Delta x_k$$
What Rudin is doing
Rather than picking a sample point in each subinterval (arbitrary), he uses sup and inf — the extremes $f$ can possibly take on each piece. $U$ is the total area if $f$ were at its highest on each piece; $L$ if at its lowest. The true area is trapped between them. The gap $U - L = \sum(M_k - m_k)\Delta x_k$ measures how much $f$ oscillates, weighted by interval width.
Definition 6.1 — Upper and Lower Integrals
$$\overline{\int_a^b} f\,dx = \inf_P\, U(P,f) \qquad \underline{\int_a^b} f\,dx = \sup_P\, L(P,f)$$ $f$ is Riemann integrable ($f\in\mathscr{R}$) if these are equal. The common value is $\int_a^b f\,dx$.
Why this is well-defined, and why it looks "backwards"
Since $f$ is assumed bounded, there exist $m \leq f(x) \leq M$ globally, so $m(b-a) \leq L(P,f) \leq U(P,f) \leq M(b-a)$ for every $P$. The set of all $U$-values is bounded below; the set of all $L$-values is bounded above. Recall Ch. 1: completeness of $\mathbb{R}$ guarantees that any nonempty bounded set has a sup/inf. So $\overline{\int}f$ and $\underline{\int}f$ always exist as finite numbers.

The inf of $U$'s is the best possible overestimate — the lowest ceiling. The sup of $L$'s is the best possible underestimate — the highest floor. Integrability means ceiling meets floor.
Check Your Understanding
T/FIt is possible for $U(P,f) < \int_a^b f\,dx$ for some partition $P$.
False
Every $U(P,f)$ is an overestimate: $U(P,f) \geq \overline{\int}f = \int f$ (when $f\in\mathscr{R}$). The upper integral is the infimum of all upper sums, so no individual $U(P,f)$ can fall below $\int f$. Symmetrically, no $L(P,f)$ can exceed $\int f$.
MCWhy does Rudin use $M_k = \sup f$ and $m_k = \inf f$ on each subinterval rather than picking a sample point $t_k \in [x_{k-1},x_k]$?
A
With sample points, the sum $\sum f(t_k)\Delta x_k$ depends on which $t_k$ you pick — different choices may give different values. Rudin avoids this by always taking the extremes. The result is the tightest possible overestimate ($U$) and underestimate ($L$) for that partition, with no degrees of freedom.
T/FThe upper and lower integrals $\overline{\int}f$ and $\underline{\int}f$ are defined for every bounded function $f$, whether or not $f\in\mathscr{R}$.
True
Boundedness of $f$ means all $U$- and $L$-values lie in $[m(b-a),\,M(b-a)]$, so the set of upper sums is bounded below and the set of lower sums is bounded above. Completeness of $\mathbb{R}$ guarantees $\inf U$ and $\sup L$ exist as finite numbers regardless of whether they are equal. Integrability is the additional requirement that they are equal.

02 Refinements

Theorem 6.4 — Refinement Lemma
If $P^*$ is a refinement of $P$ (i.e. $P \subseteq P^*$), then $L(P,f) \leq L(P^*,f) \leq U(P^*,f) \leq U(P,f)$.

Adding a point splits one subinterval into two. On each piece the sup can only be $\leq$ the original sup, so $U$ decreases (or stays). Symmetrically $L$ increases.
Corollary 6.5 — Any $L \leq$ any $U$
For any two partitions $P_1, P_2$: $L(P_1,f) \leq U(P_2,f)$.

Proof: $P^* = P_1 \cup P_2$ is a refinement of both, so $L(P_1) \leq L(P^*) \leq U(P^*) \leq U(P_2)$.

This is why $\overline{\int}f \geq \underline{\int}f$ always — the inf of all $U$'s is never below the sup of all $L$'s.
Check Your Understanding
T/FAdding a point to partition $P$ to get $P^*$ can only decrease $U(P,f)$ or keep it the same — it can never increase it.
True
Adding a point splits one subinterval $[x_{k-1},x_k]$ into two pieces. The sup of $f$ on each smaller piece is $\leq$ the sup on the whole interval. So each contributing term to $U$ can only shrink. The Refinement Lemma (6.4) captures this: $U(P^*,f) \leq U(P,f)$.
MCCorollary 6.5 says any lower sum is $\leq$ any upper sum, even for completely unrelated partitions. What is the key step in the proof?
B
$P^* = P_1 \cup P_2$ refines both $P_1$ and $P_2$. The Refinement Lemma gives $L(P_1) \leq L(P^*) \leq U(P^*) \leq U(P_2)$. Chaining these inequalities yields the corollary. This is also the key structural reason that $\underline{\int}f \leq \overline{\int}f$ always holds.

03 The Riemann Criterion

Theorem 6.6 — Riemann Criterion
$f\in\mathscr{R}$ if and only if for every $\varepsilon > 0$ there exists a partition $P$ with $U(P,f) - L(P,f) < \varepsilon$.
Why this is the tool you actually use
The definition of integrability ($\overline{\int}f = \underline{\int}f$) involves an inf and sup over all partitions — hard to compute directly. The criterion replaces that with a single partition you construct. Every integrability proof in PS8 works by exhibiting such a $P$ and bounding the gap.

PS8 Ex.1 template: uniform partition $P_n$ on $[0,c]$, gap $= c^2/n \to 0$. Then $U(P_n) \to c^2/2$ and $L(P_n) \to c^2/2$ squeeze the integral to $c^2/2$.
Check Your Understanding
T/FTo prove $f \in \mathscr{R}$ using the Riemann criterion, you must find a partition $P$ that works for all $\varepsilon > 0$ simultaneously.
False
For each $\varepsilon > 0$ you find a (possibly different) partition $P_\varepsilon$ with $U(P_\varepsilon,f) - L(P_\varepsilon,f) < \varepsilon$. The partition is allowed to depend on $\varepsilon$. In PS8 Ex.1, the uniform partition $P_n$ with $n > c^2/\varepsilon$ works — a finer partition for smaller $\varepsilon$.
MCIn PS8 Ex.1, you compute $\int_0^c x\,dx = c^2/2$ using the Riemann criterion. What quantity controls how small you can make $U(P_n,f) - L(P_n,f)$ for the uniform partition $P_n$ with $n$ subintervals?
C
For $f(x) = x$ on $[0,c]$ with uniform partition $P_n$ (mesh $c/n$): on each piece $[x_{k-1},x_k]$, $M_k = x_k = kc/n$ and $m_k = x_{k-1} = (k-1)c/n$, so $M_k - m_k = c/n$. Summing over $n$ pieces of width $c/n$: $U - L = n \cdot (c/n) \cdot (c/n) = c^2/n$. Choose $n > c^2/\varepsilon$ to make this $< \varepsilon$.

04 Which Functions Are Integrable?

Theorem 6.8 — Continuous ⟹ Integrable
$f$ continuous on $[a,b]$ implies $f\in\mathscr{R}$.

Recall Ch. 4: continuous on a compact set $\Rightarrow$ uniformly continuous. So given $\varepsilon$, choose $\delta$ with $|x-y|<\delta \Rightarrow |f(x)-f(y)|<\varepsilon/(b-a)$. Take any $P$ with $\Delta x_k < \delta$: then $M_k - m_k < \varepsilon/(b-a)$ on every piece, giving $U-L < \varepsilon$.
Theorem 6.9 — Monotone ⟹ Integrable
$f$ monotone on $[a,b]$ implies $f\in\mathscr{R}$, even if $f$ has jump discontinuities.

For increasing $f$: on the uniform partition $P_n$, $M_k = f(x_k)$ and $m_k = f(x_{k-1})$, so the sum telescopes: $U - L = \tfrac{b-a}{n}(f(b)-f(a)) \to 0$.
Non-example — Dirichlet function $f = \mathbf{1}_\mathbb{Q}$
On every subinterval, $\mathbb{Q}$ and $\mathbb{R}\setminus\mathbb{Q}$ are both dense (recall Ch. 1), so $M_k = 1$ and $m_k = 0$ for every $k$, every $P$. Thus $U(P,f) = b-a$ and $L(P,f) = 0$ always — the gap never shrinks. $f\notin\mathscr{R}$.
Check Your Understanding
T/FA function must be continuous on $[a,b]$ to be Riemann integrable on $[a,b]$.
False
Continuity implies integrability (Theorem 6.8), but not the other way. Monotone functions (Theorem 6.9) can have jump discontinuities at countably many points and still be integrable. The real condition, captured by Lebesgue's criterion, is that the set of discontinuities has measure zero — not that there are no discontinuities.
MCThe proof that continuous $\Rightarrow$ integrable (Theorem 6.8) uses uniform continuity, not just pointwise continuity. Why does the distinction matter?
B
The proof picks $\delta$ so that $|x-y| < \delta \Rightarrow |f(x)-f(y)| < \varepsilon/(b-a)$. This $\delta$ must work everywhere simultaneously — for all $k$ and all pairs of points within any subinterval of width $< \delta$. That's exactly uniform continuity. With only pointwise continuity, you get a separate $\delta_x$ at each point, and as $x$ varies these might shrink to zero, making a uniform partition useless. Recall Ch. 4: continuous on a compact set (here $[a,b]$) automatically gives uniform continuity.
T/FThe Dirichlet function $f = \mathbf{1}_\mathbb{Q}$ fails to be integrable because $\mathbb{Q}$ is countably infinite, whereas a function with only finitely many discontinuities would be integrable.
False
The Dirichlet function fails because it is discontinuous everywhere — not just at the rationals. At every irrational $x$, nearby rationals make $f$ jump, so $f$ is also discontinuous at irrationals. The discontinuity set is all of $[a,b]$, which has measure $b-a > 0$. The issue isn't countability of $\mathbb{Q}$ — it's that neither $\mathbb{Q}$ nor $\mathbb{R}\setminus\mathbb{Q}$ can be avoided in any subinterval (density from Ch. 1), so $M_k - m_k = 1$ always.

01 Algebraic Properties

These need proof — they're not obvious from the definition
In calculus you wrote $\int(f+g) = \int f + \int g$ without thought. Here you have to prove it because the integral is defined via inf/sup of sums, not a limit. The key inequality is $\sup(f+g) \leq \sup f + \sup g$ — NOT equality, because the sups might be achieved at different points. This inequality gives $U(P,f+g) \leq U(P,f) + U(P,g)$, and similarly $L(P,f+g) \geq L(P,f) + L(P,g)$. The squeeze from both sides gives additivity.

The trap on PS8 Ex.2 is the scalar case $c < 0$. When $c < 0$: $\sup(cf) = c \cdot \inf f$ (multiplying by a negative number flips the sup to an inf). So $U(P, cf) = c \cdot L(P, f)$ — the upper sum for $cf$ is $c$ times the LOWER sum for $f$. If you write $U(P,cf) = cU(P,f)$ and forget to flip, your proof breaks for $c < 0$.
Linearity, Monotonicity, Triangle Inequality (Rudin §6.12)
For $f, g \in \mathscr{R}[a,b]$ and $c \in \mathbb{R}$: $$\int_a^b (f+g) = \int_a^b f + \int_a^b g \qquad \int_a^b cf = c\int_a^b f$$ If $f \leq g$ on $[a,b]$: $\int_a^b f \leq \int_a^b g$.   And: $\left|\int_a^b f\right| \leq \int_a^b |f|$.
Monotonicity
If $f \leq g$ on $[a,b]$, then $\int_a^b f \leq \int_a^b g$.

Proof: $h = g-f \geq 0$ implies $m_k(h) \geq 0$, so $L(P,h) \geq 0$ for all $P$, giving $\int h \geq 0$.
Triangle Inequality for Integrals
If $f \in \mathscr{R}[a,b]$, then $|f| \in \mathscr{R}[a,b]$ and $\left|\int_a^b f\right| \leq \int_a^b |f|$.

Why $|f| \in \mathscr{R}$: $\big||f(x)|-|f(y)|\big| \leq |f(x)-f(y)|$, so oscillation of $|f|$ is no worse than $f$.
Why the bound: From $-|f| \leq f \leq |f|$, monotonicity gives $-\int|f| \leq \int f \leq \int|f|$.
Check Your Understanding
T/FFor bounded functions $f$ and $g$ on $[x_{k-1},x_k]$: $\sup(f+g) = \sup f + \sup g$.
False — equality fails in general
The correct inequality is $\sup(f+g) \leq \sup f + \sup g$. Equality would require both sups to be attained at the same point, which needn't happen. This is exactly why $U(P,f+g) \leq U(P,f)+U(P,g)$ (not equality) — and why you need to squeeze from both $U$ and $L$ sides to prove additivity.
MCFor a constant $c < 0$ and $f \in \mathscr{R}[a,b]$, what is the upper Darboux sum $U(P, cf)$ on a partition $P$?
C — the sup flips when you multiply by a negative
When $c < 0$: $\sup(cf) = c\inf f$ on each subinterval (multiplying by a negative number reverses the ordering). So $M_k(cf) = c\cdot m_k(f)$, and $U(P,cf) = \sum c\cdot m_k(f)\Delta x_k = c\cdot L(P,f)$. This is the hidden trap in PS8 Ex.2 — if you write $U(P,cf) = cU(P,f)$ for $c < 0$, your proof of linearity is wrong.
T/FIf $f \in \mathscr{R}[a,b]$ and $f(x) \geq 0$ for all $x$, then $\int_a^b f\,dx \geq 0$.
True — monotonicity with $g \equiv 0$
Apply monotonicity: $f \geq 0 = g$ on $[a,b]$ gives $\int f \geq \int g = 0$. Equivalently, $f \geq 0$ implies $m_k \geq 0$ on every subinterval, so $L(P,f) \geq 0$ for all $P$, hence $\underline{\int}f \geq 0$. Since $f \in \mathscr{R}$, $\int f = \underline{\int}f \geq 0$.

02 Fundamental Theorem of Calculus

FTC Part 1 and Part 2 are NOT the same theorem — know the difference
Part 1: You start with an integrable $f$ (no antiderivative needed, no continuity required). You build the function $F(x) = \int_a^x f(t)\,dt$. Claim: $F$ is always continuous, and if $f$ happens to be continuous at $x_0$, then $F'(x_0) = f(x_0)$. The hypothesis is on $f$'s continuity; the conclusion is about $F$'s differentiability.

Part 2: You start with a known antiderivative $F$ (a function you already have, with $F' = f$ on $(a,b)$). Claim: $\int_a^b f = F(b) - F(a)$. No continuity hypothesis on $f$ needed — just that $f \in \mathscr{R}$ and $F$ is continuous on $[a,b]$ with $F' = f$ on the open interval.

Where students go wrong: Part 1 does NOT say $F' = f$ everywhere — only at points where $f$ is continuous. If $f$ has a jump discontinuity at $x_0$, then $F$ is still continuous there (it's Lipschitz) but $F'(x_0)$ may not exist. The F built in Part 1 and the F assumed in Part 2 are different objects with different hypotheses.
Why $F'(x_0) = f(x_0)$ requires continuity of $f$ at $x_0$
$F(x)=\int_a^x f(t)\,dt$ measures signed area up to $x$. The difference quotient: $$\frac{F(x_0+h)-F(x_0)}{h} = \frac{1}{h}\int_{x_0}^{x_0+h} f(t)\,dt$$ This is the average value of $f$ on $[x_0, x_0+h]$. For this to converge to $f(x_0)$, you need $f(t)$ to be close to $f(x_0)$ for $t$ near $x_0$ — which is exactly continuity. Without continuity, the average over a shrinking interval might not converge to the pointwise value.

Example of failure: Let $f = \mathbf{1}_{[1/2, 1]}$ on $[0,1]$ — integrable but not continuous at $1/2$. Then $F(x) = \max(0, x - 1/2)$, which has a kink at $x = 1/2$. $F$ is not differentiable there, confirming that Part 1 breaks at the discontinuity.
FTC Part 1 — The integral function is differentiable (Rudin §6.20)
Let $f \in \mathscr{R}[a,b]$ and define $F(x) = \int_a^x f(t)\,dt$. Then:
· $F$ is continuous on $[a,b]$.
· If $f$ is continuous at $x_0$, then $F'(x_0) = f(x_0)$.

Proof sketch: $\frac{F(x_0+h)-F(x_0)}{h} - f(x_0) = \frac{1}{h}\int_{x_0}^{x_0+h}(f(t)-f(x_0))\,dt$. By continuity of $f$ at $x_0$, the integrand is $< \varepsilon$ for $|h| < \delta$, so the whole expression is $< \varepsilon$.
FTC Part 2 — Evaluation using antiderivatives (Rudin §6.21)
If $f \in \mathscr{R}[a,b]$ and $F: [a,b] \to \mathbb{R}$ is continuous with $F' = f$ on $(a,b)$, then: $$\int_a^b f(x)\,dx = F(b) - F(a)$$ Reading this: FTC Part 2 applies to ANY antiderivative $F$, not just the one defined by Part 1. This is what makes it useful for computation.
Common error on FTC Part 1
FTC Part 1 does NOT say $F'(x_0) = f(x_0)$ for all $x_0$ — only at points where $f$ is continuous. If $f$ has a jump at $x_0$, $F$ is still continuous there but may fail to be differentiable. $f \in \mathscr{R}$ is not enough to guarantee differentiability of $F$.
Check Your Understanding
T/FIf $f \in \mathscr{R}[a,b]$, the function $F(x) = \int_a^x f(t)\,dt$ is differentiable at every point of $[a,b]$.
False — only where $f$ is continuous
$F$ is always continuous (it's Lipschitz: $|F(x)-F(y)| \leq M|x-y|$ where $M = \sup|f|$). But differentiability requires the extra condition that $f$ is continuous at $x_0$. Example: $f = \mathbf{1}_{[1/2,1]}$ on $[0,1]$ gives $F(x) = \max(0, x-1/2)$, which has a corner at $x = 1/2$ — continuous but not differentiable there.
T/FFTC Part 2 only applies when $F$ is specifically defined by $F(x) = \int_a^x f(t)\,dt$.
False — any antiderivative works
FTC Part 2 applies to ANY $F$ satisfying: (1) $F$ continuous on $[a,b]$, and (2) $F' = f$ on the open interior $(a,b)$. This is what makes Part 2 useful for computation — you can use $F(x) = x^3/3$ to evaluate $\int_0^1 x^2\,dx$, no need to go back to Darboux sums. The $F$ in Part 1 and the $F$ in Part 2 are conceptually different objects.
MCWhy is $F(x) = \int_a^x f(t)\,dt$ always continuous, even when $f$ has jump discontinuities?
B — Lipschitz continuity
Since $f$ is bounded ($|f| \leq M$), for any $x < y$: $|F(y)-F(x)| = |\int_x^y f\,dt| \leq M(y-x)$. This is the definition of Lipschitz continuity with constant $M$, which implies uniform continuity. Crucially, this requires only boundedness of $f$, not continuity — so $F$ is continuous regardless of where $f$ jumps.

03 Integration by Parts & Substitution

Integration by Parts (Rudin §6.22)
Let $u, v: [a,b] \to \mathbb{R}$ be continuously differentiable. Then: $$\int_a^b u'(x)v(x)\,dx = u(b)v(b) - u(a)v(a) - \int_a^b u(x)v'(x)\,dx$$ Proof: Apply FTC to the product rule $(uv)' = u'v + uv'$, then rearrange.
Integration by Substitution (Rudin §6.19)
Let $\varphi: [a,b] \to \mathbb{R}$ be $C^1$ and $f$ continuous on the range of $\varphi$. Then: $$\int_a^b f(\varphi(x))\,\varphi'(x)\,dx = \int_{\varphi(a)}^{\varphi(b)} f(t)\,dt$$ Proof: Let $F$ be an antiderivative of $f$. Chain rule gives $(F\circ\varphi)' = f(\varphi(x))\varphi'(x)$. Apply FTC Part 2 to both sides.
Exam trap — hypotheses matter
Substitution requires $f$ continuous on the range of $\varphi$ AND $\varphi$ continuously differentiable — not just differentiable. Always state these hypotheses explicitly when writing proofs.
Check Your Understanding
MCIntegration by parts is derived by applying FTC to which differentiation rule?
B — product rule reversed
$(uv)' = u'v + uv'$. Integrate both sides over $[a,b]$ via FTC: $[uv]_a^b = \int u'v + \int uv'$. Rearrange to isolate $\int u'v$. Substitution instead comes from the chain rule: $(F\circ\varphi)' = f(\varphi(x))\varphi'(x)$, with FTC applied to the left side. Knowing which rule is being reversed tells you which formula you get.
T/FThe substitution formula requires $\varphi \in C^1$ — a merely differentiable $\varphi$ (with discontinuous derivative) is not sufficient.
True — $\varphi'$ must be continuous (or at least integrable)
The proof uses FTC Part 2 applied to $F\circ\varphi$ on $[a,b]$, which requires $(F\circ\varphi)' = f(\varphi(x))\varphi'(x)$ to be integrable. If $\varphi'$ is discontinuous, $f(\varphi(x))\varphi'(x)$ may not be integrable, and FTC Part 2 cannot be invoked. $C^1$ (continuous derivative) guarantees $\varphi' \in \mathscr{R}$, making the formula valid. Always state this hypothesis explicitly in proofs.

01 Measure Zero and Lebesgue's Criterion

Main Idea
The exact boundary of Riemann integrability: a bounded function is integrable if and only if its set of discontinuities has measure zero. This unifies and explains all the integrability theorems.
Definition — Measure Zero (PS8 Ex.4)
$A \subseteq \mathbb{R}$ has measure zero if for every $\varepsilon > 0$, there exist open intervals $\{O_n\}$ with $A \subseteq \bigcup O_n$ and $\sum \ell(O_n) < \varepsilon$.

Finite sets are measure zero. Countable sets are measure zero. The Cantor set is measure zero despite being uncountable.
Lebesgue's Criterion — complete characterization (PS8 Ex.4)
A bounded $f: [a,b] \to \mathbb{R}$ is Darboux integrable $\iff$ the set of discontinuities of $f$ has measure zero.

Why this matters: Explains why continuous $f$ (zero discontinuities) and monotone $f$ (at most countably many discontinuities = measure zero) are integrable, while the Dirichlet function (discontinuous everywhere — discontinuity set $= [a,b]$, measure $b-a > 0$) is not.
Proof strategy for PS8 Ex.4
⇒
Define $D_\alpha = \{x : \omega(f,x) \geq \alpha\}$ where $\omega(f,x)$ is the oscillation at $x$. Show $D_\alpha$ is compact. By the Riemann criterion, choose $P$ with $U-L < \alpha\varepsilon$. Intervals meeting $D_\alpha$ have $M_k - m_k \geq \alpha$, so their total length $\leq \varepsilon$.
Then: $D = \bigcup D_{1/n}$ is a countable union of measure-zero sets, hence measure zero.
⇐
Cover discontinuity set $D$ by open intervals of total length $< \varepsilon/(4M)$. Let $K = [a,b] \setminus \text{cover}$. $K$ is compact and $f|_K$ is uniformly continuous. Handle $K$ by uniform continuity; handle the covering by bounding oscillation by $2M$. Combine to get $U-L < \varepsilon$.
Key: splitting into "good" (near $K$) and "bad" (near $D$) subintervals.
Check Your Understanding
T/FEvery countable subset of $\mathbb{R}$ has measure zero.
True — cover the $n$-th point by a shrinking interval
Let $A = \{a_1, a_2, \ldots\}$ be countable. Given $\varepsilon > 0$, cover $a_n$ by the open interval $(a_n - \varepsilon/2^{n+1},\, a_n + \varepsilon/2^{n+1})$, which has length $\varepsilon/2^n$. The total length is $\sum_{n=1}^\infty \varepsilon/2^n = \varepsilon$. So $A$ has measure zero. This is why countably many discontinuities (e.g., a monotone function's jumps) pose no obstacle to integrability.
T/FThe Cantor set has positive measure because it is uncountable.
False — measure and cardinality are independent
The Cantor set is uncountable AND has measure zero. Measure zero means the set can be covered by open intervals of arbitrarily small total length — this has nothing to do with how many points the set contains. The Cantor set has measure zero because at each stage of its construction you remove open intervals; the total length removed is $1/3 + 2/9 + 4/27 + \cdots = 1$, leaving a set of measure $0$.
MCLebesgue's criterion immediately explains why monotone functions are integrable. What is the reason?
C — countable discontinuities = measure zero
A monotone function can have jump discontinuities (countably many — each jump has positive height, and infinitely many positive-height jumps would force the function to be unbounded). Since countable sets have measure zero (shown above), Lebesgue's criterion immediately gives $f \in \mathscr{R}$. This is more elegant than the direct proof via telescoping sums, and shows that Thm 6.9 is just a special case of the general criterion.

02 Hölder's Inequality

Hölder's Inequality (PS8 Ex.6)
Let $f, g \in \mathscr{R}[a,b]$, $p, q > 1$ with $\frac{1}{p}+\frac{1}{q}=1$. Then: $$\left|\int_a^b fg\,dx\right| \leq \left(\int_a^b |f|^p\right)^{1/p}\left(\int_a^b |g|^q\right)^{1/q}$$ The case $p = q = 2$ is the Cauchy-Schwarz inequality.
Proof — via Young's inequality
1
Young's inequality: $ab \leq a^p/p + b^q/q$ for $a,b \geq 0$.
Proof: fix $b$, minimize $g(a) = a^p/p + b^q/q - ab$ over $a \geq 0$. The minimum is $0$.
2
Let $A = \|f\|_p$ and $B = \|g\|_q$. If either is $0$, done. Otherwise normalize: $\tilde{f} = |f|/A$, $\tilde{g} = |g|/B$, so $\int\tilde{f}^p = \int\tilde{g}^q = 1$.
This reduces to the case where both $L^p$-norms equal $1$.
3
Apply Young's pointwise: $\tilde{f}(x)\tilde{g}(x) \leq \tilde{f}^p/p + \tilde{g}^q/q$. Integrate: $\int\tilde{f}\tilde{g} \leq 1/p + 1/q = 1$. Unscale: $\int|fg| \leq AB$. Then $|\int fg| \leq \int|fg| \leq AB$.
Check Your Understanding
T/FThe Cauchy-Schwarz inequality $\left|\int_a^b fg\right| \leq \sqrt{\int_a^b f^2}\cdot\sqrt{\int_a^b g^2}$ is a special case of Hölder's inequality.
True — take $p = q = 2$
With $p = q = 2$: $1/p + 1/q = 1/2 + 1/2 = 1$ ✓. Hölder gives $|\int fg| \leq \|f\|_2\|g\|_2 = (\int f^2)^{1/2}(\int g^2)^{1/2}$, which is exactly Cauchy-Schwarz. The Hölder inequality is the $L^p$-generalization of Cauchy-Schwarz to all conjugate pairs $p,q > 1$.

04 Fundamental Lemma of the Calculus of Variations

Fundamental Lemma (PS8 Ex.8)
Let $f: [a,b] \to \mathbb{R}$ be continuous. If $\int_a^b f(x)\phi(x)\,dx = 0$ for every continuous $\phi: [a,b] \to \mathbb{R}$, then $f \equiv 0$.
Proof by contradiction
1
Suppose $f(x_0) > 0$ for some $x_0$. By continuity, $f > 0$ on some interval $I = (x_0-\delta, x_0+\delta) \cap [a,b]$.
2
Choose a bump function: $\phi(x) = \max(0, \delta - |x-x_0|)$ (a tent function). This is continuous, zero outside $I$, positive inside $I$.
Test function technique — ubiquitous in PDEs and distribution theory.
3
Then $f(x)\phi(x) > 0$ on $I$ and $\geq 0$ elsewhere. So $\int_a^b f\phi > 0$ — contradiction. $\blacksquare$
Check Your Understanding
T/FThere is a one-line proof of the fundamental lemma: since $f$ is continuous (hence $\phi = f$ is an allowed test function), taking $\phi = f$ in the hypothesis gives $\int_a^b f^2 = 0$; since $f^2 \geq 0$ and $f$ is continuous, this forces $f \equiv 0$.
True — this is a valid and elegant proof
Since $f$ is continuous, $\phi = f$ is a legal test function. The hypothesis $\int f\phi = 0$ for all continuous $\phi$ then gives $\int_a^b f^2\,dx = 0$. Since $f^2 \geq 0$ and $f^2$ is continuous, $\int f^2 = 0$ forces $f^2 \equiv 0$ on $[a,b]$ — if $f^2(x_0) > 0$ at any point, continuity makes $f^2 > 0$ on a neighborhood, giving a positive integral. So $f \equiv 0$. The tent-function proof works more generally; this shortcut uses the specific hypothesis.
MCWhy does the fundamental lemma require $f$ to be continuous? What goes wrong if $f$ is only assumed integrable?
C — "almost everywhere" is the best without continuity
If $f$ is integrable but not continuous, changing $f$ on a single point (or any measure-zero set) doesn't change any integral. So $\int f\phi = 0$ for all continuous $\phi$ only pins down $f$ up to measure-zero modification. For example, $f = \mathbf{1}_{\{c\}}$ (the indicator of a single point) satisfies $\int f\phi = 0$ for all $\phi$, yet $f \not\equiv 0$. Continuity rules this out: if $f(x_0) \neq 0$ and $f$ is continuous, the value propagates to a whole interval with positive measure.

PS8 Ex.2Let $f,g \in \mathscr{R}[a,b]$. Prove linearity, monotonicity, and the triangle inequality for the Riemann integral.PS8

These feel obvious from calculus, but here you must prove them from the definition, because the integral is defined via inf/sup of sums — not a limit. The hidden trap is the scalar case $c < 0$: since $\sup(cf) = c\inf f$ when $c < 0$ (multiplying by a negative flips sup to inf), the upper sum for $cf$ equals $c$ times the lower sum for $f$. Writing $U(P,cf) = cU(P,f)$ for negative $c$ is the most common error here.

Approach
1. Additivity: Use $\sup(f{+}g) \leq \sup f + \sup g$ to get $U(P,f{+}g) \leq U(P,f)+U(P,g)$. Choose $P$ making each within $\varepsilon/2$ of its integral, then squeeze.

4. Triangle inequality: First prove $|f| \in \mathscr{R}$ via $\sup|f| - \inf|f| \leq \sup f - \inf f$ (reverse triangle inequality). Then $-|f| \leq f \leq |f|$ and monotonicity give $-\int|f| \leq \int f \leq \int|f|$.
Solution sketch
Part 1 (Additivity): $U(P,f+g) \leq U(P,f)+U(P,g)$ and $L(P,f+g) \geq L(P,f)+L(P,g)$. Given $\varepsilon > 0$, choose $P$ so each of $U(P,f), U(P,g)$ is within $\varepsilon/2$ of its integral. Part 2: For $c \geq 0$: $\sup(cf) = c\sup f$, so $U(P,cf) = cU(P,f)$. For $c < 0$: $\sup(cf) = c\inf f = |c|\cdot(-\inf f)$, giving $U(P,cf) = -|c|L(P,f)$. Infimize/supremize to get $c\int f$ in both cases.

Rudin 6.10Let $p \geq 0$, $p \in \mathscr{R}$, and $f$ continuous on $[a,b]$. Show $\exists\,c \in [a,b]$ with $\int_a^b fp = f(c)\int_a^b p$.RUDIN

This is a weighted mean value theorem for integrals. Think of $p$ as a weight function — since $p \geq 0$, the integral $\int fp$ is a weighted average of $f$, and any weighted average of a continuous function must equal $f$ at some interior point $c$ (by IVT applied to $f$, which is continuous on $[a,b]$). Watch the edge case: if $\int p = 0$, then since $p \geq 0$, we have $p \equiv 0$ and both sides are trivially zero.

Approach
1. Let $m = \min f$, $M = \max f$ (attained since $f$ is continuous on compact $[a,b]$, by EVT from Ch. 4).
2. Since $p \geq 0$: $m \cdot p(x) \leq f(x)p(x) \leq M \cdot p(x)$ pointwise. Integrate to get $m\int p \leq \int fp \leq M\int p$.
3. If $\int p = 0$: $p \equiv 0$, so both sides are $0$. If $\int p > 0$: divide to get $m \leq \frac{\int fp}{\int p} \leq M$. By IVT on $f$, there exists $c$ with $f(c) = \frac{\int fp}{\int p}$.
Solution
Let $m = \min f$, $M = \max f$ (attained since $f$ continuous on compact $[a,b]$). Since $p \geq 0$ and $m \leq f \leq M$: $m\int p \leq \int fp \leq M\int p$.

If $\int p = 0$: then $p \equiv 0$ (since $p \geq 0$), so $\int fp = 0 = f(c)\int p$ for any $c$. If $\int p > 0$: $m \leq \frac{\int fp}{\int p} \leq M$. By IVT applied to $f$, $\exists\,c$ with $f(c) = \frac{\int fp}{\int p}$. $\square$

T/FA refinement of a partition $P$ increases $U(f,P)$ and decreases $L(f,P)$.

T/FIf $f$ is monotone on $[a,b]$, then $f \in \mathscr{R}[a,b]$.

Recall Rudin Thm 6.9: for increasing $f$, on a uniform partition of $n$ pieces $M_k = f(x_k)$ and $m_k = f(x_{k-1})$, so the sum $U - L$ telescopes. The key insight is that jump discontinuities don't block integrability — all that matters is the total variation $f(b)-f(a)$, which is finite.

True — Rudin Thm 6.9
Uniform partition of $n$ pieces: $U-L = \frac{b-a}{n}(f(b)-f(a)) \to 0$. Regardless of how many jump discontinuities $f$ has, the total variation $f(b)-f(a)$ is bounded and can be spread over $n$ pieces.
T/FFor any bounded $f:[a,b]\to\mathbb{R}$ and any partition $P$: $L(f,P) \leq U(f,P)$.

On any subinterval, $\inf f \leq \sup f$ is an elementary fact about infima and suprema — it has nothing to do with integrability. Ask yourself: does this inequality need $f \in \mathscr{R}$ to hold, or is it true for every bounded function on every partition?

True — always, no integrability required
On each subinterval: $m_k = \inf f \leq \sup f = M_k$, so $\sum m_k\Delta x_k \leq \sum M_k\Delta x_k$. This holds for any bounded $f$ and any partition. The stronger statement: $L(f,P_1) \leq U(f,P_2)$ for ANY two partitions (proved via common refinement).
