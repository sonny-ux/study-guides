Vector Fields · Line Integrals · FTC & Conservative Fields
Definition & Motivation
Vector Field

A vector field on a domain $D \subseteq \mathbb{R}^n$ is a function $\vec{F}: D \to \mathbb{R}^n$ that assigns a vector to each point. In two dimensions:

Definition
$\vec{F}(x,y) = \langle P(x,y),\, Q(x,y) \rangle$ where $P, Q: D \to \mathbb{R}$ are the component functions.

In three dimensions: $\vec{F}(x,y,z) = \langle P, Q, R \rangle = P\,\vec{i} + Q\,\vec{j} + R\,\vec{k}$.

Physical Contexts

Vector fields arise naturally throughout physics:

Velocity field
Fluid flow: at each point $(x,y)$, $\vec{F}(x,y)$ gives the velocity of the fluid particle there.
Force field
Gravity, electric, magnetic fields — at each point, $\vec{F}$ gives the force on a test mass/charge.
Standard Examples
Key Examples from Lecture
  • 1$\vec{F} = \langle x, y \rangle$ — radial outward: vectors point directly away from the origin, magnitude grows with distance.
  • 2$\vec{F} = \langle -y, x \rangle$ — rotation: vectors are tangent to circles centered at the origin (counterclockwise).
  • 3$\vec{F}(x,y,z) = z\,\vec{k}$ — purely vertical field, strength proportional to height.
  • 4$\vec{F}(x,y,z) = \dfrac{-GMm}{|\vec{x}|^3}\vec{x}$ where $\vec{x} = \langle x,y,z \rangle$ — gravitational force field, inverse-square law.
Gradient Fields & Conservative Fields
Gradient Vector Field

If $f: D \to \mathbb{R}$ is a scalar function, its gradient

Gradient field
$$\nabla f = \left\langle \frac{\partial f}{\partial x},\, \frac{\partial f}{\partial y} \right\rangle$$

is itself a vector field. Geometrically, $\nabla f$ at each point is perpendicular to the level sets $\{f = c\}$ and points in the direction of steepest ascent.

Conservative Fields
Definition
$\vec{F}$ is conservative if there exists a scalar function $f$ (called a potential) such that $\vec{F} = \nabla f$.

Key observation: If $\vec{F} = \nabla f$, then $\vec{F}$ is orthogonal to the level sets $\{f = c\}$ at every point.

Caution
Being orthogonal to level curves does not imply $\vec{F}$ is conservative! Consider:
$\vec{F} = \dfrac{1}{x^2+y^2}\langle -y, x \rangle$

This is orthogonal to all rays through the origin, but it is not conservative — the domain $\mathbb{R}^2\setminus\{0\}$ has a hole, which matters (see Lecture 9).

E7.1

Find the gradient vector field $\nabla f$ for $f(x,y) = \tfrac{1}{2}(x-y)^2$. Describe its behavior along the line $x = y$, and on either side of it.
Solution

$\partial_x f = (x-y)$, $\quad \partial_y f = -(x-y) = y-x$.

So $\nabla f(x,y) = \langle x-y,\; y-x \rangle = (x-y)\langle 1,-1\rangle$.

On $x=y$: $\nabla f = \vec{0}$. For $x>y$: arrows point in direction $\langle 1,-1\rangle$ (down-right). For $x < y$: arrows point $\langle -1,1\rangle$ (up-left). All arrows are parallel — the field has constant direction, varying magnitude.

E7.2

Find the gradient vector field $\nabla f$ for $f(x,y) = \tfrac{1}{2}(x^2 - y^2)$. Describe the pattern geometrically.
Solution

$\partial_x f = x$, $\quad \partial_y f = -y$.

So $\nabla f(x,y) = \langle x, -y\rangle$.

Arrows point away from the $y$-axis horizontally and toward the $x$-axis vertically — a saddle pattern. The level curves of $f$ are hyperbolas $x^2-y^2=c$, and $\nabla f$ is orthogonal to them.

S1

Is $\vec{F}(x,y) = \langle y^2,\; 2xy + 1\rangle$ conservative? If so, find a potential $f$.
Solution

Check: $\partial_y(y^2) = 2y$ and $\partial_x(2xy+1) = 2y$. Equal — so $\vec{F}$ is conservative on $\mathbb{R}^2$.

From $\partial_x f = y^2$: integrate in $x$ to get $f = xy^2 + g(y)$.

From $\partial_y f = 2xy + g'(y) = 2xy+1$: $g'(y)=1$, so $g(y) = y$.

Potential: $f(x,y) = xy^2 + y$.

S2

Conceptual. Suppose $\vec{F} = \nabla f$ where $f(x,y) = x^2 + y^2$. Without computing, describe what the vector field looks like and what its level sets are. Why is $\vec{F}$ orthogonal to the circles $x^2+y^2 = c$?
Solution

$\nabla f = \langle 2x, 2y\rangle$ — a radially outward field, proportional to the position vector. Arrows get longer farther from the origin.

Level sets are circles $x^2+y^2=c$. The gradient always points in the direction of steepest increase of $f$, which is radially outward — perpendicular to the circular level curves. This is the general fact: $\nabla f \perp \{f=c\}$.

HW16.1.19

Match the vector field $\vec{F}(x,y) = \langle 1, \sin y \rangle$ with the correct plot. Describe the behavior: how do the vectors change as $y$ varies, and what happens along horizontal lines?
Solution

Analyze the components. The $x$-component is always $1$ — every vector has the same horizontal length. The $y$-component is $\sin y$, which oscillates between $-1$ and $1$ depending on $y$.

Behavior along horizontal lines ($y = c$). On any horizontal strip where $y = c$, all vectors are identical: $\langle 1, \sin c \rangle$. The field is independent of $x$.

Key features. Along $y = 0, \pm\pi, \ldots$: vectors are $\langle 1, 0\rangle$ — purely horizontal. Along $y = \pi/2 + 2k\pi$: vectors tilt upward to $\langle 1, 1\rangle$. Along $y = -\pi/2 + 2k\pi$: vectors tilt downward to $\langle 1, -1\rangle$. The pattern repeats periodically in $y$ with no dependence on $x$. This matches the plot showing horizontal periodicity of the vertical tilt.

HW16.1.20

Match the vector field $\vec{F}(x,y) = \langle x, -y \rangle$ with the correct plot. Describe the behavior near the origin, along the axes, and generally.
Solution

Behavior on the axes. On the $x$-axis ($y=0$): $\vec{F} = \langle x, 0\rangle$ — vectors point away from the origin horizontally. On the $y$-axis ($x=0$): $\vec{F} = \langle 0, -y\rangle$ — vectors point toward the origin vertically.

General behavior. The field stretches horizontally and compresses vertically, pointing toward the $x$-axis from above and below. The integral curves are hyperbolas $xy = c$ (solutions to $dx/dt = x$, $dy/dt = -y$ give $xe^{-t}$ growing, $ye^t$ decaying). This is a saddle or hyperbolic flow pattern.

Origin is a fixed point. For large $|x|$ with $y$ moderate, the $x$-component dominates. This field matches the plot showing outward horizontal stretch with inward vertical compression.

HW16.1.21

Match the vector field $\vec{F}(x,y) = \langle y - x, x + y \rangle$ with the correct plot. What type of flow does this produce?
Solution

At the origin: $\vec{F}(0,0) = \langle 0, 0\rangle$ — fixed point.

On positive $x$-axis ($y=0$, $x>0$): $\vec{F} = \langle -x, x\rangle$ — vectors point up-left (counterclockwise rotation). On positive $y$-axis ($x=0$, $y>0$): $\vec{F} = \langle y, y\rangle$ — vectors point up-right.

Structure. Write in matrix form: $\vec{F} = \begin{pmatrix}-1 & 1\\ 1 & 1\end{pmatrix}\begin{pmatrix}x\\y\end{pmatrix}$. This is a linear field with both rotation and expansion. The magnitude $|\vec{F}| = \sqrt{(y-x)^2+(x+y)^2} = \sqrt{2(x^2+y^2)}$ grows with distance from the origin — vectors spiral outward. This matches the plot showing a counterclockwise expanding spiral.

HW16.1.22

Match the vector field $\vec{F}(x,y) = \langle \cos(x+y), x \rangle$ with the correct plot. Describe how the vectors vary.
Solution

$x$-component: $\cos(x+y)$ oscillates between $-1$ and $1$ depending on both $x$ and $y$ — it's constant along the diagonal lines $x+y = c$.

$y$-component: $x$ increases as we move right and is independent of $y$. So vectors tilt more upward to the right and more downward to the left, with no $y$-dependence in this component.

Combined behavior. The horizontal component creates oscillating bands (constant along diagonals), while the vertical component creates a left-right asymmetry. On the $y$-axis ($x=0$): $\vec{F} = \langle \cos y, 0\rangle$ — purely horizontal, oscillating. This matches the plot showing diagonal banding with a systematic left-right tilt.

HW16.1.26

Determine whether $\vec{F}(x,y) = \langle ye^x + \sin y,\; e^x + x\cos y \rangle$ is conservative. If so, find a potential function $f$ such that $\nabla f = \vec{F}$.
Solution

Step 1 — Curl test. Let $P = ye^x + \sin y$ and $Q = e^x + x\cos y$. $$\frac{\partial P}{\partial y} = e^x + \cos y, \qquad \frac{\partial Q}{\partial x} = e^x + \cos y$$ Equal on all of $\mathbb{R}^2$ (simply connected) — so $\vec{F}$ is conservative.

Step 2 — Integrate $f_x = P$. $$f(x,y) = \int (ye^x + \sin y)\,dx = ye^x + x\sin y + g(y)$$

Step 3 — Use $f_y = Q$ to find $g(y)$. $$f_y = e^x + x\cos y + g'(y) = e^x + x\cos y \implies g'(y) = 0 \implies g(y) = C$$

Potential function: $$\boxed{f(x,y) = ye^x + x\sin y + C}$$

HW16.1.28

Determine whether $\vec{F}(x,y) = \langle 2xy + y^{-2},\; x^2 - 2xy^{-3} \rangle$ is conservative (for $y \neq 0$). If so, find a potential function.
Solution

Step 1 — Curl test. $P = 2xy + y^{-2}$, $Q = x^2 - 2xy^{-3}$. $$\frac{\partial P}{\partial y} = 2x - 2y^{-3}, \qquad \frac{\partial Q}{\partial x} = 2x - 2y^{-3}$$ Equal wherever $y \neq 0$ — so $\vec{F}$ is conservative on the half-plane $y > 0$ (or $y < 0$).

Step 2 — Integrate $f_x = P$. $$f = \int (2xy + y^{-2})\,dx = x^2 y + xy^{-2} + g(y)$$

Step 3 — Use $f_y = Q$. $$f_y = x^2 - 2xy^{-3} + g'(y) = x^2 - 2xy^{-3} \implies g'(y) = 0 \implies g = C$$

Potential function: $$\boxed{f(x,y) = x^2 y + xy^{-2} + C}$$

HW16.1.31

Find the gradient vector field $\nabla f$ of $f(x,y) = \ln(x + 2y)$.
Solution

$$f_x = \frac{1}{x+2y}, \qquad f_y = \frac{2}{x+2y}$$

$$\nabla f(x,y) = \left\langle \frac{1}{x+2y},\; \frac{2}{x+2y} \right\rangle$$

The field is defined for $x + 2y > 0$. Note that $\nabla f = \frac{1}{x+2y}\langle 1, 2\rangle$ — all arrows are parallel to $\langle 1, 2\rangle$, with magnitude decreasing as $x+2y$ grows. The level curves of $f$ are the lines $x + 2y = c$, and $\nabla f$ is indeed orthogonal to them.

HW16.1.32

Find the gradient vector field $\nabla f$ of $f(x,y) = \tan(3x - 4y)$.
Solution

$$f_x = 3\sec^2(3x-4y), \qquad f_y = -4\sec^2(3x-4y)$$

$$\nabla f(x,y) = \sec^2(3x-4y)\langle 3,\, -4\rangle$$

All arrows are parallel to $\langle 3, -4\rangle$ (pointing in the direction of steepest increase of $\tan$). The magnitude $|\nabla f| = 5\sec^2(3x-4y)$ blows up on the lines $3x - 4y = \pi/2 + k\pi$ where $\tan$ has vertical asymptotes.

HW16.1.33

Find the gradient vector field $\nabla f$ of $f(x,y,z) = \sqrt{x^2 + y^2 + z^2}$.
Solution

Let $r = \sqrt{x^2+y^2+z^2}$. By the chain rule: $$f_x = \frac{x}{\sqrt{x^2+y^2+z^2}} = \frac{x}{r}, \quad f_y = \frac{y}{r}, \quad f_z = \frac{z}{r}$$

$$\nabla f = \frac{1}{r}\langle x, y, z\rangle = \frac{\vec{r}}{|\vec{r}|}$$

This is the unit radial field — every vector points directly away from the origin with magnitude $1$ (except at the origin itself). The level surfaces of $f$ are spheres $r = c$, and $\nabla f$ is perpendicular to them, as expected.

HW16.1.34

Find the gradient vector field $\nabla f$ of $f(x,y,z) = x^2 y e^{yz}$.
Solution

Compute each partial: $$f_x = 2xye^{yz}$$ $$f_y = x^2 e^{yz} + x^2 y \cdot ze^{yz} = x^2 e^{yz}(1 + yz)$$ $$f_z = x^2 y \cdot ye^{yz} = x^2 y^2 e^{yz}$$

$$\nabla f = e^{yz}\langle 2xy,\; x^2(1+yz),\; x^2 y^2\rangle$$

Type 1: Scalar Line Integrals $\int_C f\,ds$
First: what is $\vec{\gamma}(t)$ and why do we need it?
The parametrization idea
A curve $C$ in the plane is a shape — it doesn't come with numbers attached. To integrate along it, we need to turn it into something a calculus formula can chew on. That's what parametrization does.

You introduce a variable $t$ (think of it as time, or just a label) that sweeps from $a$ to $b$ as you travel along the curve. At each value of $t$, you record where you are: $$\vec{\gamma}(t) = \bigl(x(t),\; y(t)\bigr)$$ So $\vec{\gamma}(t)$ is just a pair of functions that give your $x$ and $y$ coordinates at "time" $t$. When $t = a$ you're at the start of the curve; when $t = b$ you're at the end.
What is $\vec{\gamma}'(t)$ and what does $|\vec{\gamma}'(t)|$ mean?
Since $\vec{\gamma}(t) = (x(t), y(t))$ gives your position at each $t$, you can ask: how is my position changing as $t$ increases? That's just ordinary differentiation of each coordinate: $$\vec{\gamma}'(t) = \bigl(x'(t),\, y'(t)\bigr)$$ So $\vec{\gamma}'(t)$ is a pair of numbers — the rate $x$ is changing, and the rate $y$ is changing, at that moment.

Now, $|\vec{\gamma}'(t)| = \sqrt{x'(t)^2 + y'(t)^2}$ is just the Pythagorean length of that pair. It answers the question: if I move a tiny step $dt$ in parameter space, how long a piece of actual curve do I trace out?

Think of it this way: if $x$ is barely changing but $y$ is changing fast, you're moving mostly vertically — and the arc length you cover per unit of $t$ is approximately $|y'(t)|$. The full formula $\sqrt{x'^2 + y'^2}$ handles the general case where both coordinates are changing at once. Multiply by $dt$ and you get the tiny arc length: $ds = \sqrt{x'(t)^2 + y'(t)^2}\,dt$.
The formula, decoded
Scalar line integral formula
$$\int_C f\,ds = \int_a^b \underbrace{f(\vec{\gamma}(t))}_{\text{value of }f\text{ at your location}} \cdot \underbrace{|\vec{\gamma}'(t)|}_{\text{your speed}} \,dt$$

In words: at each moment $t$, evaluate $f$ at your current position $\vec{\gamma}(t)$, multiply by your speed $|\vec{\gamma}'(t)|$ (which converts the time step $dt$ into an arc-length step $ds$), and integrate over $t$ from $a$ to $b$.

Physical picture: imagine $C$ is a metal wire and $f(x,y)$ is the density of the wire at each point. Then $\int_C f\,ds$ is the total mass. If $f = 1$ everywhere, you just get the total arc length of the wire.

INTERACTIVE — DRAG TO MOVE ALONG THE CURVE
Direction doesn't matter here
If you walk the curve backwards, your speed $|\vec{\gamma}'|$ is still positive — it just points the other way — so the integral is unchanged: $\int_{-C} f\,ds = \int_C f\,ds$. (This will not be true for the vector line integrals in Types 2 and 3.)

Piecewise curves: if $C$ is made of several pieces, just add the integrals: $\displaystyle\int_{C_1 \cup C_2} f\,ds = \int_{C_1} f\,ds + \int_{C_2} f\,ds$

Worked Example — $\int_C 2x\,ds$, piecewise curve

Evaluate $\int_C 2x\,ds$ where $C = C_1 \cup C_2$: $C_1$ is the parabola $y = x^2$ from $(0,0)$ to $(1,1)$, and $C_2$ is the vertical segment from $(1,1)$ to $(1,2)$.

Step 0 — how to parametrize each piece
For $C_1$ (the parabola $y = x^2$): let $t$ be the $x$-coordinate itself, running from $0$ to $1$. Then $x(t) = t$ and $y(t) = t^2$, so $\vec{\gamma}_1(t) = (t,\, t^2)$.

For $C_2$ (the vertical segment $x=1$, $y$ from $1$ to $2$): let $t$ run from $0$ to $1$. Then $x(t) = 1$ (constant) and $y(t) = 1 + t$, so $\vec{\gamma}_2(t) = (1,\, 1+t)$.
  • C₁ speed Differentiate: $\vec{\gamma}_1'(t) = (x'(t),\, y'(t)) = (1,\, 2t)$.
    Speed: $|\vec{\gamma}_1'(t)| = \sqrt{1^2 + (2t)^2} = \sqrt{1 + 4t^2}$.
    This is just the Pythagorean theorem on the velocity components.
  • C₁ integral On $C_1$, $x = t$, so $f = 2x = 2t$. Plug everything in: $$\int_{C_1} 2x\,ds = \int_0^1 2t \cdot \sqrt{1+4t^2}\,dt$$ Substitute $u = 1+4t^2$, $du = 8t\,dt$: $$= \tfrac{1}{4}\int_1^5 \sqrt{u}\,du = \tfrac{1}{4}\cdot\tfrac{2}{3}\bigl[u^{3/2}\bigr]_1^5 = \tfrac{1}{6}(5\sqrt{5}-1)$$
  • C₂ speed Differentiate: $\vec{\gamma}_2'(t) = (0,\, 1)$.
    Speed: $|\vec{\gamma}_2'(t)| = \sqrt{0^2 + 1^2} = 1$.
    Makes sense — you're moving at constant speed straight up, no $x$-motion.
  • C₂ integral On $C_2$, $x = 1$ (constant), so $f = 2x = 2$. Then: $$\int_{C_2} 2x\,ds = \int_0^1 2 \cdot 1\,dt = 2$$
  • Total $$\int_C 2x\,ds = \frac{5\sqrt{5}-1}{6} + 2$$
Type 2: Integrals w.r.t. $dx$, $dy$
What does $dx$ or $dy$ even mean along a curve?
The idea
In a regular integral $\int f(x)\,dx$, the $dx$ is a tiny step in the $x$-direction. Along a curve, as you move a tiny time step $dt$, your $x$-coordinate changes by $dx = x'(t)\,dt$ and your $y$-coordinate changes by $dy = y'(t)\,dt$. So writing $\int_C P\,dx + Q\,dy$ means: at each point of the curve, take $P$ times the tiny $x$-change, plus $Q$ times the tiny $y$-change, and add all of it up.

To actually compute this, you parametrize the curve as $\vec{\gamma}(t) = (x(t), y(t))$, substitute $dx = x'(t)\,dt$ and $dy = y'(t)\,dt$, and everything becomes a normal single-variable integral in $t$:

Formula — decoded piece by piece
$$\int_C P\,dx + Q\,dy = \int_a^b \Bigl[\underbrace{P(x(t),y(t))}_{\text{evaluate }P\text{ at your location}} \cdot \underbrace{x'(t)}_{\substack{\text{rate of change} \\ \text{of }x}} + \underbrace{Q(x(t),y(t))}_{\text{evaluate }Q\text{ at your location}} \cdot \underbrace{y'(t)}_{\substack{\text{rate of change} \\ \text{of }y}}\Bigr]\,dt$$

So the recipe is always the same: (1) write a parametrization $(x(t), y(t))$, (2) compute $x'(t)$ and $y'(t)$, (3) substitute into the formula, (4) integrate in $t$.

Direction matters here — unlike Type 1
If you reverse the path, $x'(t)$ and $y'(t)$ both flip sign (you're going the other way), so the whole integral flips sign: $\int_{-C} P\,dx + Q\,dy = -\int_C P\,dx + Q\,dy$. This is different from Type 1 where speed $|\vec{\gamma}'|$ is always positive.
Worked Example — $\int_C y^2\,dx + x\,dy$, two different paths

We'll integrate $P = y^2$, $Q = x$ along two different paths connecting $(-5,-3)$ to $(0,2)$. The punchline: the answers are different, which means this field is not conservative.

How to read the integrand $y^2\,dx + x\,dy$
At each point of the curve, you're computing: ($y^2$ times the tiny horizontal step) + ($x$ times the tiny vertical step). The relative sizes of those steps depend on the direction you're traveling, which is why the path shape matters.
  • Path (i) — line segment Parametrize a straight line from $(-5,-3)$ to $(0,2)$.
    A line segment from point $A$ to point $B$ can always be written as $A + t(B - A)$ for $t \in [0,1]$. Here:
    $x(t) = -5 + 5t, \quad y(t) = -3 + 5t$
    Check: at $t=0$, $(x,y) = (-5,-3)$ ✓. At $t=1$, $(x,y) = (0,2)$ ✓.

    Compute derivatives: $x'(t) = 5$, $\quad y'(t) = 5$.

    Substitute $P = y^2$ and $Q = x$:
    $P = y^2 = (-3+5t)^2$, $\quad Q = x = -5+5t$

    Plug into the formula: $$\int_0^1 \bigl[(-3+5t)^2 \cdot 5 + (-5+5t)\cdot 5\bigr]\,dt = 5\int_0^1\bigl[(5t-3)^2 + (5t-5)\bigr]\,dt$$ Expand $(5t-3)^2 = 25t^2 - 30t + 9$ and add $(5t-5)$: $$= 5\int_0^1(25t^2 - 25t + 4)\,dt = 5\Bigl[\tfrac{25t^3}{3} - \tfrac{25t^2}{2} + 4t\Bigr]_0^1 = 5\Bigl(\tfrac{25}{3} - \tfrac{25}{2} + 4\Bigr) = \boxed{-\tfrac{5}{6}}$$
  • Path (ii) — arc Parametrize the curve $x = 4 - y^2$ from $(-5,-3)$ to $(0,2)$.
    Since $x$ is given as a function of $y$, use $y$ itself as the parameter: let $t = y$, running from $-3$ to $2$. Then $x(t) = 4 - t^2$ and $y(t) = t$.
    Check: at $t=-3$, $(x,y) = (4-9,-3) = (-5,-3)$ ✓. At $t=2$, $(x,y) = (0,2)$ ✓.

    Compute derivatives: $x'(t) = -2t$, $\quad y'(t) = 1$.

    Substitute $P = y^2 = t^2$ and $Q = x = 4 - t^2$: $$\int_{-3}^{2}\bigl[t^2\cdot(-2t) + (4-t^2)\cdot 1\bigr]\,dt = \int_{-3}^{2}(-2t^3 - t^2 + 4)\,dt$$ $$= \Bigl[-\tfrac{t^4}{2} - \tfrac{t^3}{3} + 4t\Bigr]_{-3}^{2} = \Bigl(-8 - \tfrac{8}{3} + 8\Bigr) - \Bigl(-\tfrac{81}{2} + 9 - 12\Bigr) = \boxed{\tfrac{245}{6}}$$
Why the answers are different
Path (i) gives $-5/6$. Path (ii) gives $245/6$. Same start, same end, completely different answers. This is the hallmark of a field that is not conservative — the path you take genuinely matters. If it were conservative, both paths would give the same number.
Type 3: Vector Line Integrals $\int_C \vec{F}\cdot d\vec{r}$
Work Done by a Force Field

Partition $C$ into small displacement vectors $\vec{\gamma}(t_i)-\vec{\gamma}(t_{i-1}) \approx \vec{\gamma}'(t_i^*)\Delta t_i$. Work $\approx \sum \vec{F}(\vec{\gamma}(t_i^*))\cdot\vec{\gamma}'(t_i^*)\,\Delta t_i$. Taking the limit:

Formula
$$\int_C \vec{F}\cdot d\vec{r} = \int_a^b \vec{F}(\vec{\gamma}(t))\cdot\vec{\gamma}'(t)\,dt$$

Writing $d\vec{r} = \langle dx, dy, dz \rangle$, this is the same as $\int_C F_x\,dx + F_y\,dy + F_z\,dz$.

Orientation dependence
$\displaystyle\int_{-C}\vec{F}\cdot d\vec{r} = -\int_C \vec{F}\cdot d\vec{r}$. Reversing the path reverses the sign (unlike scalar integrals).
Worked Example — $\vec{F} = \langle xy, yz, zx \rangle$, twisted cubic

Let $\vec{\gamma}(t) = \langle t, t^2, t^3 \rangle$, $0 \le t \le 1$.

  • 1$\vec{\gamma}'(t) = \langle 1, 2t, 3t^2 \rangle$
  • 2$\vec{F}(\vec{\gamma}(t)) = \langle t\cdot t^2,\; t^2\cdot t^3,\; t^3\cdot t \rangle = \langle t^3, t^5, t^4 \rangle$
  • 3Dot product: $t^3(1) + t^5(2t) + t^4(3t^2) = t^3 + 2t^6 + 3t^6 = t^3 + 5t^6$
  • 4$\displaystyle\int_0^1 (t^3 + 5t^6)\,dt = \frac{1}{4} + \frac{5}{7} = \frac{27}{28}$

E8.1

Evaluate $\int_C (x+2y)\,dx + x^2\,dy$ where $C$ is the piecewise linear path $(0,0)\to(2,1)\to(3,0)$.
Solution

$C_1$: $(0,0)\to(2,1)$. Parametrize $\vec{r}_1(t)=\langle 2t,t\rangle$, $0\le t\le 1$. So $dx=2\,dt$, $dy=dt$, $x+2y=4t$, $x^2=4t^2$.

$\displaystyle\int_{C_1} = \int_0^1(8t+4t^2)\,dt = \Bigl[4t^2+\tfrac{4}{3}t^3\Bigr]_0^1 = \tfrac{16}{3}$

$C_2$: $(2,1)\to(3,0)$. Parametrize $\vec{r}_2(t)=\langle 2+t,1-t\rangle$. Then $x+2y=4-t$ and $x^2=(2+t)^2$, $dy=-dt$.

$\displaystyle\int_{C_2} = \int_0^1\bigl[(4-t)-(4+4t+t^2)\bigr]\,dt = \int_0^1(-5t-t^2)\,dt = -\tfrac{17}{6}$

Total: $\tfrac{16}{3} - \tfrac{17}{6} = \boxed{\tfrac{5}{2}}$.

E8.2

Evaluate $\int_C z^2\,dx + x^2\,dy + y^2\,dz$ where $C$ is the line segment from $(1,0,0)$ to $(4,1,2)$.
Solution

Parametrize: $\vec{r}(t)=\langle 1+3t,\, t,\, 2t\rangle$, $0\le t\le 1$. So $dx=3\,dt$, $dy=dt$, $dz=2\,dt$.

Integrand: $(4t^2)(3)+(1+3t)^2(1)+(t^2)(2) = 12t^2+1+6t+9t^2+2t^2 = 1+6t+23t^2$

$\displaystyle\int_0^1(1+6t+23t^2)\,dt = \Bigl[t+3t^2+\tfrac{23}{3}t^3\Bigr]_0^1 = 1+3+\tfrac{23}{3} = \tfrac{35}{3}$

E8.3

Conceptual. (a) Show a constant force $\vec{F}=\langle a,b\rangle$ does zero work around $x^2+y^2=1$. (b) Same for $\vec{F}(\vec{x})=k\vec{x}$?
Solution

(a) Parametrize: $\vec{r}(t)=\langle\cos t,\sin t\rangle$, $\vec{r}'=\langle-\sin t,\cos t\rangle$. Work $=\int_0^{2\pi}\!\bigl(a(-\sin t)+b\cos t\bigr)dt=[a\cos t+b\sin t]_0^{2\pi}=0$.

(b) Yes. For $\vec{F}=k\langle\cos t,\sin t\rangle$: integrand $=k\cos t(-\sin t)+k\sin t(\cos t)=0$ identically. Alternatively, $\vec{F}=\nabla\!\bigl(\tfrac{k}{2}(x^2+y^2)\bigr)$ is conservative $\Rightarrow\oint=0$.

Evaluate the scalar line integral $\int_C (x^2+y)\,ds$ where $C$ is the quarter circle $x^2+y^2=4$ from $(2,0)$ to $(0,2)$.

Fundamental Theorem for Line Integrals
What this theorem actually says
The big idea
Remember the single-variable FTC: $\int_a^b f'(x)\,dx = f(b)-f(a)$. You don't care what $f'$ does in between — only the values at the two endpoints matter. This theorem says the exact same thing works for line integrals, as long as the field is a gradient.

If your vector field $\vec{F}$ happens to be the gradient of some scalar function $f$ — i.e. $\vec{F} = \nabla f$ — then integrating $\vec{F}$ along any path from point $A$ to point $B$ gives you exactly $f(B) - f(A)$. The shape of the path is completely irrelevant.

Theorem
If $\vec{F} = \nabla f$ (i.e. $\vec{F}$ is conservative with potential $f$) and $C$ goes from $\vec{\gamma}(a)$ to $\vec{\gamma}(b)$, then: $$\int_C \vec{F}\cdot d\vec{r} = \int_C \nabla f\cdot d\vec{r} = f(\vec{\gamma}(b)) - f(\vec{\gamma}(a))$$
Why it works — chain rule argument
The key is that $\vec{F}\cdot d\vec{r} = \nabla f(\vec{\gamma}(t))\cdot\vec{\gamma}'(t)\,dt$. But by the multivariable chain rule, that dot product is exactly $\tfrac{d}{dt}f(\vec{\gamma}(t))$. So the integral becomes: $$\int_a^b \frac{d}{dt}f(\vec{\gamma}(t))\,dt = f(\vec{\gamma}(b)) - f(\vec{\gamma}(a))$$ That last step is just the ordinary single-variable FTC. $\square$
What $\nabla f \cdot d\vec{r}$ means in words
$d\vec{r}$ is a tiny step along the curve. $\nabla f$ at that point tells you the direction $f$ increases fastest. Their dot product is "how much does $f$ change during this tiny step?" Summing all those tiny changes along the curve just gives you the net change: $f(\text{end}) - f(\text{start})$.

TRUE

This is exactly the FTC for line integrals: $\int_C\nabla f\cdot d\vec{r} = f(\text{end})-f(\text{start})$. The path shape is irrelevant — only where you start and stop matters.

Path independence in action — all paths between two points give the same value for a conservative field. So swap a nasty spiral for a straight line or axis-aligned path. (See E9.4: the parabolic path got replaced by the $x$-axis.)

The Big Equivalence: Three Ways to Say the Same Thing
Three equivalent conditions for a vector field

For a vector field $\vec{F}$ on a connected open domain $D$, these three statements either all hold or all fail together. They are completely interchangeable — proving any one of them proves all three.

  • Zero on closed loops. $\displaystyle\oint_C \vec{F}\cdot d\vec{r} = 0$ for every closed curve $C$ in $D$. In plain English: if you start somewhere, travel any loop, and return to where you started, the total work done by $\vec{F}$ is zero. Like hiking a circular trail — you end up at the same elevation you started.
  • Path independence. $\displaystyle\int_{C_1} \vec{F}\cdot d\vec{r} = \int_{C_2} \vec{F}\cdot d\vec{r}$ whenever $C_1$ and $C_2$ share the same start and end. In plain English: it doesn't matter which route you take between two cities — the work done by $\vec{F}$ is the same every time.
  • Conservative — has a potential. $\vec{F} = \nabla f$ for some scalar function $f$. In plain English: $\vec{F}$ is secretly just the gradient of a "height map" $f$. Integrating $\vec{F}$ between two points is the same as reading off the height difference.
Proof sketch — how the three imply each other
(1)⟹(2): If $C_1$ and $C_2$ share endpoints, build a closed loop: go along $C_1$, then backwards along $C_2$. That loop has zero integral by (1), so $\int_{C_1} = \int_{C_2}$.

(2)⟹(3): Fix a base point $q_0$. Define $f(q) = \int_{C_q}\vec{F}\cdot d\vec{r}$ along any path from $q_0$ to $q$ — path independence (2) guarantees this is well-defined, no matter which path you choose. Then one shows $\nabla f = \vec{F}$.

(3)⟹(1): By FTC, any closed loop from $p$ back to $p$ gives $f(p)-f(p)=0$.
Quick Check — True or False
T/F

If $\int_C \vec{F}\cdot d\vec{r} = 0$ for some particular closed curve $C$, then $\vec{F}$ is conservative.
FALSE

Zero on one closed curve is not enough. The equivalence theorem requires zero on every closed curve in the domain. A non-conservative field can easily give zero on a specific well-chosen loop.

T/F

If $\vec{F}$ is conservative and $C$ is any closed curve, then $\int_C \vec{F}\cdot d\vec{r} = 0$ regardless of the shape or size of $C$.
TRUE

Direct from FTC: $C$ starts and ends at the same point $p$, so $\int_C\nabla f\cdot d\vec{r} = f(p)-f(p) = 0$. Doesn't matter how wild the path is — start equals end every time.

The Curl Test — How to Check Without Finding a Potential
The quick screening test ($\partial_y P = \partial_x Q$)
What does $\partial_y P = \partial_x Q$ actually mean?
Write $\vec{F} = \langle P(x,y),\; Q(x,y)\rangle$. The condition says: take the $x$-component $P$ and differentiate it with respect to $y$, then take the $y$-component $Q$ and differentiate it with respect to $x$ — if you're conservative, these must be equal.

Why? Because if $\vec{F} = \nabla f$, then $P = \partial_x f$ and $Q = \partial_y f$. So $\partial_y P = \partial_y\partial_x f$ and $\partial_x Q = \partial_x\partial_y f$ — and mixed partials are equal (Clairaut's theorem). If they're not equal, $\vec{F}$ is definitively not conservative. Done.
The test, precisely
If $\vec{F} = \langle P, Q \rangle$ is conservative on $D$, then everywhere on $D$: $$\partial_y P = \partial_x Q$$ Contrapositive: if $\partial_y P \neq \partial_x Q$ at even one point, $\vec{F}$ is not conservative.
The converse — when the test also works backwards
If $\partial_y P = \partial_x Q$ on a simply connected domain $D$, then $\vec{F}$ is conservative.

Simply connected just means the domain has no holes. Think of a region you could shrink any rubber-band loop inside of without it getting stuck on anything. The whole plane $\mathbb{R}^2$, a disk, a rectangle — all simply connected. An annulus (donut shape) or $\mathbb{R}^2$ minus the origin — not simply connected.
Why "no holes" matters — the standard counterexample
$\vec{F} = \dfrac{1}{x^2+y^2}\langle -y,\,x\rangle$ satisfies $\partial_y P = \partial_x Q$ everywhere it's defined. But it's defined on $\mathbb{R}^2\setminus\{0\}$, which has a hole at the origin. And in fact $\vec{F}$ is not conservative: integrating around the unit circle gives $2\pi \neq 0$. The hole at the origin "traps" a non-zero circulation that the curl test can't detect.
Examples — applying the test
  • Not conservative$\vec{F} = \langle x-y,\; x-2 \rangle$: $\partial_y(x-y) = -1$ but $\partial_x(x-2) = +1$. They differ, so $\vec{F}$ is not conservative. You could stop here — no potential exists, path independence fails.
  • Conservative$\vec{F} = \langle ye^x + \sin y,\; e^x + x\cos y \rangle$: $\partial_y(ye^x+\sin y) = e^x+\cos y$ and $\partial_x(e^x+x\cos y) = e^x+\cos y$. Equal on $\mathbb{R}^2$ (no holes), so $\vec{F}$ is conservative — a potential exists.
Quick Check — True or False
T/F

If $\partial_y P = \partial_x Q$ everywhere on a region $D$, then $\vec{F} = \langle P,Q\rangle$ must be conservative on $D$.
FALSE

The curl test is necessary but not always sufficient. The converse only holds on simply connected domains (no holes). Counterexample: $\vec{F} = \frac{1}{x^2+y^2}\langle -y,x\rangle$ passes the curl test on $\mathbb{R}^2\setminus\{0\}$ but is not conservative — the missing origin is a hole that breaks things.

Fix: add "on a simply connected domain."

T/F

If $\vec{F} = \langle P,Q\rangle$ is conservative, it is necessary that $\partial_y P = \partial_x Q$ on its domain.
TRUE

The curl test gives a necessary condition. If $\vec{F} = \nabla f$, then $P = \partial_x f$ and $Q = \partial_y f$, and Clairaut's theorem forces $\partial_y P = \partial_y\partial_x f = \partial_x\partial_y f = \partial_x Q$. So if the test fails, the field is immediately not conservative — cheap to check, can save a lot of work.

Finding the Potential Function $f$
What you're actually doing

If $\vec{F}=\langle P,Q\rangle$ is conservative, you know $\partial_x f = P$ and $\partial_y f = Q$ for some mystery function $f$. Your job is to reverse-engineer $f$. Think of it as running integration twice, where the "constant of integration" is actually a function of the other variable that you have to pin down.

Method 1: Partial Integration (most common)
  • 1Integrate one component. Take $\partial_x f = P$ and integrate w.r.t. $x$ — but because $y$ is a free variable, the "constant" of integration can actually be a function $h(y)$: $f = \int P\,dx + h(y)$.
  • 2Pin down $h(y)$. Differentiate your $f$ w.r.t. $y$, set it equal to $Q$, and solve for $h'(y)$. Then integrate to get $h(y)$.
Example — $\vec{F} = \langle ye^x + \sin y,\; e^x + x\cos y \rangle$
  • 1Integrate the $x$-component w.r.t. $x$: $f = ye^x + x\sin y + h(y)$. Here $h(y)$ is our unknown "constant".
  • 2Differentiate w.r.t. $y$: $\partial_y f = e^x + x\cos y + h'(y)$. Set equal to $Q = e^x + x\cos y$: this forces $h'(y)=0$, so $h(y)=K$ (a true constant).
  • 3Potential: $f(x,y) = ye^x + x\sin y + K$.
Method 2: Star-Shaped Recipe (elegant, always works on hole-free domains)

Instead of solving partial differential equations, just evaluate one integral along the straight line from the origin to your point $\vec{q}$. This is the formula the professor derives from the path independence construction:

Formula
$$f(\vec{q}) = \int_0^1 \vec{F}(t\vec{q})\cdot\vec{q}\,dt$$ where the path is $\vec{\gamma}(t) = t\vec{q}$, a straight line from the origin to $\vec{q}$.
Example — $\vec{F} = \langle 3+2xy,\; x^2-3y^2 \rangle$, find $f(u,v)$
  • 1Plug in $t\vec{q}=(tu,tv)$: $\vec{F}(tu,tv) = \langle 3+2uvt^2,\; u^2t^2-3v^2t^2 \rangle$
  • 2Dot with $\vec{q}=(u,v)$: $(3+2uvt^2)u + (u^2t^2-3v^2t^2)v = 3u + 3u^2vt^2 - 3v^3t^2$
  • 3Integrate from $0$ to $1$: $f(u,v) = 3u + u^2v - v^3$, i.e. $f(x,y) = 3x + x^2y - y^3$

Verify: $\partial_x(3x+x^2y-y^3)=3+2xy$ ✓ and $\partial_y(3x+x^2y-y^3)=x^2-3y^2$ ✓

Quick Check — True or False
T/F

A conservative vector field has exactly one potential function.
FALSE

Potentials are unique only up to a constant. If $f$ is a potential, so is $f + C$ for any constant $C$ — just like $+C$ in single-variable antiderivatives. Adding a constant doesn't change $\nabla f$, so it remains a valid potential.

Example — Scary spiral path, trivial answer

Evaluate $\int_C \vec{F}\cdot d\vec{r}$ where $\vec{F} = \langle 3+2xy,\; x^2-3y^2 \rangle$ and $\vec{\gamma}(t) = \langle e^t\sin t,\; e^t\cos t \rangle$, $0 \le t \le \pi$.

The curve is a decaying spiral — trying to integrate this directly would be a nightmare. But we don't have to.

  • 1From Method 2 above, $\vec{F}$ is conservative with potential $f(x,y) = 3x + x^2y - y^3$.
  • 2Start point: $\vec{\gamma}(0) = \langle e^0\sin 0, e^0\cos 0\rangle = \langle 0, 1\rangle$. Plug in: $f(0,1) = 0 + 0 - 1 = -1$.
  • 3End point: $\vec{\gamma}(\pi) = \langle e^\pi\sin\pi, e^\pi\cos\pi\rangle = \langle 0, -e^\pi\rangle$. Plug in: $f(0,-e^\pi) = 0 + 0 - (-e^\pi)^3 = e^{3\pi}$.
  • 4By FTC: $\displaystyle\int_C \vec{F}\cdot d\vec{r} = f(\text{end}) - f(\text{start}) = e^{3\pi} - (-1) = e^{3\pi}+1$.

The spiral path was completely irrelevant. Once you knew $\vec{F}$ was conservative, only the start and end coordinates mattered.

How to attack these problems
The 3-step game plan
Step 1 — Check the curl test. Compute $\partial_y P$ and $\partial_x Q$ (and the analogues in 3D). If they match everywhere on a hole-free domain, the field is conservative.

Step 2 — Find a potential $\phi$. Either integrate the first component w.r.t. $x$ and fix the "constant" $h(y)$ using the second component (partial integration), or use the star-shaped recipe $\phi(\vec{q})=\int_0^1\vec{F}(t\vec{q})\cdot\vec{q}\,dt$.

Step 3 — Plug in endpoints. $\int_C\vec{F}\cdot d\vec{r}=\phi(\text{end})-\phi(\text{start})$. The path is irrelevant — only where you start and stop matters.
In plain English
A conservative field is like elevation on a hill: no matter which route you hike from one town to another, the total change in height is the same. The potential $\phi$ is the elevation function. The line integral just becomes a subtraction.
E9.1

Find the work done by $\vec{F}(x,y)=x^3\,\vec{i}+y^3\,\vec{j}$ in moving an object from $P(1,0)$ to $Q(2,2)$. (Exercise 1 from Ex9)
Solution

Step 1 — Is it conservative? Check $\partial_y(x^3)=0$ and $\partial_x(y^3)=0$. Equal — yes, and $\mathbb{R}^2$ has no holes, so $\vec{F}$ is conservative.

Step 2 — Find potential. We need $\partial_x\phi=x^3$, so integrate: $\phi=\frac{x^4}{4}+h(y)$. Then $\partial_y\phi=h'(y)=y^3$, giving $h(y)=\frac{y^4}{4}$.

$$\phi(x,y)=\frac{x^4}{4}+\frac{y^4}{4}$$

Step 3 — Apply FTC.

$$W=\phi(2,2)-\phi(1,0)=\left(\frac{16}{4}+\frac{16}{4}\right)-\left(\frac{1}{4}+0\right)=8-\frac{1}{4}=\boxed{\frac{31}{4}}$$

No need to parametrize the path at all — only the endpoints matter.

E9.2

Let $\vec{F}(x,y,z)=2xy\,\vec{i}+(x^2+2yz)\,\vec{j}+y^2\,\vec{k}$. Find a potential $\phi$ and evaluate $\int_C\vec{F}\cdot d\vec{r}$ along the line segment from $(2,-3,1)$ to $(-5,1,2)$. (Exercise 2 from Ex9)
Solution

Finding the potential (partial integration in 3D).

Start from $\partial_x\phi=2xy$. Integrate w.r.t. $x$: $\phi=x^2y+g(y,z)$.

Now $\partial_y\phi=x^2+\partial_y g=x^2+2yz$, so $\partial_y g=2yz$, meaning $g=y^2z+h(z)$.

Finally $\partial_z\phi=y^2+h'(z)=y^2$, so $h'(z)=0$. Thus:

$$\phi(x,y,z)=x^2y+y^2z$$

(Quick sanity: $\nabla\phi=\langle 2xy,\; x^2+2yz,\; y^2\rangle=\vec{F}$ ✓)

Apply FTC.

$\phi(2,-3,1)=(4)(-3)+9(1)=-12+9=-3$

$\phi(-5,1,2)=(25)(1)+(1)(2)=25+2=27$

$$\int_C\vec{F}\cdot d\vec{r}=27-(-3)=\boxed{30}$$

Use path independence — pick the easy path $y=0$, $x$ from $0$ to $3$.

Along $y=0$: $\vec{r}(t)=\langle t,0\rangle$, $d\vec{r}=\langle 1,0\rangle dt$. Only the $x$-component of $\vec{F}$ contributes: $F_x=\sin 0+e^t=e^t$.

$$\int_C\vec{F}\cdot d\vec{r}=\int_0^3 e^t\,dt=\bigl[e^t\bigr]_0^3=\boxed{e^3-1}$$

The original path $y=t(3-t)$ would be a nightmare to integrate directly — path independence lets you swap it for the $x$-axis instead.

S1

Conceptual. $\vec{F}=\dfrac{1}{x^2+y^2}\langle -y,\,x\rangle$ satisfies $\partial_y P=\partial_x Q$ everywhere it's defined. Is it conservative? Explain why or why not.
Solution

No — it is not conservative, despite passing the curl test.

The issue is the domain: $\vec{F}$ is defined on $\mathbb{R}^2\setminus\{0\}$, which has a hole at the origin. The converse of the curl test only guarantees conservativity on simply connected domains (no holes).

To confirm it's not conservative, compute $\oint_C\vec{F}\cdot d\vec{r}$ around the unit circle $\vec{r}(t)=\langle\cos t,\sin t\rangle$, $0\le t\le 2\pi$:

$\vec{F}(\vec{r}(t))=\frac{1}{1}\langle -\sin t,\cos t\rangle$ and $\vec{r}'(t)=\langle -\sin t,\cos t\rangle$.

Integrand: $\sin^2 t+\cos^2 t=1$, so $\oint\vec{F}\cdot d\vec{r}=\int_0^{2\pi}1\,dt=2\pi\neq 0$.

A non-zero integral around a closed curve proves the field is not conservative. The hole at the origin is what breaks everything.

Quick Check — True or False
T/F

Reversing the direction of path $C$ does not change the value of $\int_C \vec{F}\cdot d\vec{r}$.
FALSE

For vector line integrals, direction matters: $\int_{-C}\vec{F}\cdot d\vec{r} = -\int_C\vec{F}\cdot d\vec{r}$. Reversing the path reverses $\vec{\gamma}'(t)$, which flips all the dot products and therefore the sign.

Contrast: scalar line integrals $\int_C f\,ds$ are direction-independent because $ds = |\vec{\gamma}'|\,dt$ is always positive. One more reason to keep track of which type of integral you're computing.

HW16.3.6

Show that $\vec{F}$ is conservative and evaluate $\displaystyle\int_C \vec{F}\cdot d\vec{r}$, where $\vec{F}(x,y) = \langle (1+xy)e^{xy},\; x^2 e^{xy}\rangle$ and $C$ is given by $\vec{r}(t) = \langle \cos t,\, 2\sin t\rangle$, $0 \le t \le \pi$.
Solution

Step 1 — Curl test. $P = (1+xy)e^{xy}$, $Q = x^2 e^{xy}$. $$\frac{\partial P}{\partial y} = xe^{xy} + x(1+xy)e^{xy} = xe^{xy}(1 + 1 + xy) = xe^{xy}(2+xy)$$ $$\frac{\partial Q}{\partial x} = 2xe^{xy} + x^2 \cdot ye^{xy} = xe^{xy}(2+xy)$$ Equal on $\mathbb{R}^2$ — so $\vec{F}$ is conservative.

Step 2 — Find a potential $f$ with $\nabla f = \vec{F}$.

Observe that $\frac{\partial}{\partial x}(xe^{xy}) = e^{xy} + xye^{xy} = (1+xy)e^{xy} = P$. So try $f(x,y) = xe^{xy}$.

Check: $f_y = x\cdot xe^{xy} = x^2 e^{xy} = Q$. ✓

So $f(x,y) = xe^{xy}$.

Step 3 — Identify endpoints. The curve $\vec{r}(t) = \langle\cos t, 2\sin t\rangle$ is the upper half of the ellipse $x^2 + y^2/4 = 1$.

At $t=0$: $\vec{r}(0) = \langle 1, 0\rangle$. At $t=\pi$: $\vec{r}(\pi) = \langle -1, 0\rangle$.

Step 4 — Apply FTC. $$\int_C \vec{F}\cdot d\vec{r} = f(-1, 0) - f(1, 0) = (-1)e^{0} - (1)e^{0} = -1 - 1 = \boxed{-2}$$

HW16.3.41

Find a function $f$ such that $\nabla f = \vec{F}$, or show that no such $f$ exists, where $\vec{F}(x,y,z) = \langle e^y,\ xe^y + e^z,\ ye^z\rangle$. Then use it to evaluate $\int_C \vec{F}\cdot d\vec{r}$ along any path from $(0,0,0)$ to $(1,1,1)$.
Solution

Step 1 — Check conservativity. $P = e^y$, $Q = xe^y + e^z$, $R = ye^z$. $$P_y = e^y = Q_x = e^y \checkmark$$ $$P_z = 0 = R_x = 0 \checkmark$$ $$Q_z = e^z = R_y = e^z \checkmark$$ All cross-partials match on $\mathbb{R}^3$ — conservative.

Step 2 — Find $f$.

From $f_x = e^y$: integrate in $x$ to get $f = xe^y + g(y,z)$.

From $f_y = xe^y + g_y = xe^y + e^z$: so $g_y = e^z$, giving $g = ye^z + h(z)$.

From $f_z = ye^z + h'(z) = ye^z$: so $h'(z) = 0$, $h = C$.

Potential: $f(x,y,z) = xe^y + ye^z$.

Step 3 — Apply FTC. $$\int_C \vec{F}\cdot d\vec{r} = f(1,1,1) - f(0,0,0) = (e + e) - (0 + 0) = \boxed{2e}$$

L7 — Vector Fields
R1

Let $f(x,y,z) = x^2y + yz^3$. Compute the gradient vector field $\nabla f$ and describe what it means geometrically at a point where $\nabla f \neq \vec{0}$.
Solution

Compute each partial:

$\partial_x f = 2xy, \quad \partial_y f = x^2 + z^3, \quad \partial_z f = 3yz^2$

$$\nabla f(x,y,z) = \langle 2xy,\; x^2+z^3,\; 3yz^2 \rangle$$

Geometrically: at any point where $\nabla f \neq \vec{0}$, the vector $\nabla f$ points in the direction of steepest increase of $f$, and is perpendicular to the level surface $\{f = c\}$ passing through that point.

R2

Is $\vec{F}(x,y) = \langle 2x\sin y,\; x^2\cos y + 2y \rangle$ conservative? If so, find a potential $\phi$.
Solution

Curl test. $\partial_y(2x\sin y) = 2x\cos y$ and $\partial_x(x^2\cos y + 2y) = 2x\cos y$. Equal on $\mathbb{R}^2$ — conservative.

Partial integration. From $\partial_x\phi = 2x\sin y$, integrate w.r.t. $x$:

$\phi = x^2\sin y + h(y)$

Differentiate w.r.t. $y$: $\partial_y\phi = x^2\cos y + h'(y) = x^2\cos y + 2y$, so $h'(y) = 2y$, giving $h(y) = y^2$.

$$\phi(x,y) = x^2\sin y + y^2$$

Verify: $\nabla\phi = \langle 2x\sin y,\; x^2\cos y+2y\rangle = \vec{F}$ ✓

L8 — Line Integrals
R3

Compute $\displaystyle\int_C (x^2 + y)\,ds$ where $C$ is the line segment from $(0,0)$ to $(3,4)$.
Solution

Parametrize: $\vec{\gamma}(t) = \langle 3t, 4t\rangle$, $0 \le t \le 1$. Then $\vec{\gamma}'(t) = \langle 3,4\rangle$, so $|\vec{\gamma}'| = 5$.

Substitute $x = 3t$, $y = 4t$: $x^2 + y = 9t^2 + 4t$.

$$\int_C(x^2+y)\,ds = \int_0^1(9t^2+4t)\cdot 5\,dt = 5\Bigl[3t^3+2t^2\Bigr]_0^1 = 5(3+2) = \boxed{25}$$

R4

Compute $\displaystyle\int_C y\,dx - x\,dy$ where $C$ is the upper semicircle $x^2+y^2=1$ traversed from $(1,0)$ to $(-1,0)$.
Solution

Parametrize: $\vec{\gamma}(t) = \langle\cos t, \sin t\rangle$, $0 \le t \le \pi$. So $dx = -\sin t\,dt$ and $dy = \cos t\,dt$.

Integrand: $y\,dx - x\,dy = \sin t(-\sin t)\,dt - \cos t(\cos t)\,dt = (-\sin^2 t - \cos^2 t)\,dt = -\,dt$.

$$\int_C y\,dx - x\,dy = \int_0^\pi (-1)\,dt = \boxed{-\pi}$$

Note: $\vec{F} = \langle y, -x\rangle$ — check the curl test: $\partial_y(y) = 1 \neq -1 = \partial_x(-x)$. Not conservative, so this integral depends on the path chosen.

L9 — FTC, Potentials & Conservative Fields
R5

Let $\vec{F}(x,y,z) = \langle yz,\; xz,\; xy \rangle$. Find a potential $\phi$ and evaluate $\displaystyle\int_C \vec{F}\cdot d\vec{r}$ along any path from $(1,2,3)$ to $(0,1,-1)$.
Solution

Find potential. From $\partial_x\phi = yz$, integrate: $\phi = xyz + g(y,z)$. Then $\partial_y\phi = xz + \partial_y g = xz$, so $\partial_y g = 0$, meaning $g = h(z)$. Then $\partial_z\phi = xy + h'(z) = xy$, so $h'(z) = 0$.

$$\phi(x,y,z) = xyz$$

Apply FTC.

$\phi(0,1,-1) = (0)(1)(-1) = 0$

$\phi(1,2,3) = (1)(2)(3) = 6$

$$\int_C\vec{F}\cdot d\vec{r} = \phi(0,1,-1) - \phi(1,2,3) = 0 - 6 = \boxed{-6}$$

R6

Let $\vec{F}(x,y) = \langle e^x\sin y,\; e^x\cos y \rangle$. Show $\vec{F}$ is conservative, find a potential, and evaluate $\displaystyle\int_C \vec{F}\cdot d\vec{r}$ along any path from $(0,0)$ to $\left(1,\tfrac{\pi}{2}\right)$.
Solution

Curl test. $\partial_y(e^x\sin y) = e^x\cos y$ and $\partial_x(e^x\cos y) = e^x\cos y$. Equal on $\mathbb{R}^2$ — conservative.

Potential. From $\partial_x\phi = e^x\sin y$: $\phi = e^x\sin y + h(y)$. Then $\partial_y\phi = e^x\cos y + h'(y) = e^x\cos y$, so $h'(y) = 0$.

$$\phi(x,y) = e^x\sin y$$

FTC.

$\phi\!\left(1,\tfrac{\pi}{2}\right) = e^1\cdot\sin\tfrac{\pi}{2} = e\cdot 1 = e$

$\phi(0,0) = e^0\cdot\sin 0 = 1\cdot 0 = 0$

$$\int_C\vec{F}\cdot d\vec{r} = e - 0 = \boxed{e}$$
